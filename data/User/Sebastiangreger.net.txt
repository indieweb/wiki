= <span class="h-card">[http://sebastiangreger.net Sebastian Greger]</span> =

My site http://sebastiangreger.net is running on WordPress, with IndieAuth and microformats set up manually. I use a modified version of the [[hum]] link shortener.

The site sends and accepts Webmentions using the [https://github.com/pfefferle/wordpress-webmention wordpress-webmention] and [https://github.com/pfefferle/wordpress-semantic-linkbacks semantic-linkbacks] plugins. The backfeed is served by [[brid.gy]].

I POSSE to Twitter at [http://twitter.com/sebastiangreger @sebastiangreger], using my own [http://sebastiangreger.net/2014/03/audience-context-conscious-posse-syndication-plugin-wordpress/ audience/context-conscious POSSE plugin] (in development) and occasionally manually to Google+.
