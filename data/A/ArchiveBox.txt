'''<dfn>ArchiveBox</dfn>''' is an Open source, self-hosted, offline first, flat file backed Python program web page archiver that produces PDF versions, screenshots, WARC versions, etc as well. It's build on top of existing solutions such as Google Chrome's PDF output, wget, youtube-dl, etc. See [https://archivebox.io/].
