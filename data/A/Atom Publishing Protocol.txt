    '''<dfn>Atom Publishing Protocol</dfn>''' is an IETF standard API for [[CRUD]] operations between blogging client and server software, RFC 5023, and supported by [[Blogger]] and [[MarsEdit]].

== Clients ==
* [[MarsEdit]]

== Services ==
* [[Blogger]]

== Servers ==
No known server software supports AtomPub.

== History ==
AtomPub was adopted by Blogger to replace their original XML-RPC Blogger API.
