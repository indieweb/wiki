'''<dfn>[https://github.com/capjamesg/aurora Aurora]</dfn>''' is an open source [[static site generator]] developed by {{capjamesg}} and used to build [https://jamesg.blog James' Coffee Blog].

[https://jamesg.blog/aurora See the Aurora documentation].

James is developing Aurora v2 which is incompatible with v1 but substantially simpler in terms of the code used (and therefore easier to maintain). This work is happening at https://codeberg.org/capjamesg/aurora-v2/.

== Features ==

* Generates pages fast (1,500 pages in < 5s).
* Generate static sites from markdown and HTML files.
* Use [[Jinja]] templating to write logic.
* Use front matter to store metadata.
