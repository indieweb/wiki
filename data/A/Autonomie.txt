'''<dfn>[https://github.com/pfefferle/Autonomie Autonomie]</dfn>''' is a highly semantic, responsive, accessible and search engine optimized WordPress Theme. It provides HTML5 templates refined with [[microformats]], [[microformats2]] and [[microdata]] (Schema.org). Autonomie supports a lot of OpenWeb plugins and is fully IndieWeb compatible. 

The theme is inspired by Medium, supports the new Block Editor (Gutenberg) and is based on the code of the [[SemPress]] theme.

== Features ==

=== WebSemantics ===

Autonomies code is marked-up with microformats and microdata:

* used [http://microformats.org/ microformats]:
** [http://microformats.org/wiki/hatom hAtom]
** [http://microformats.org/wiki/hcard hCard]
** [http://microformats.org/wiki/rel-tag rel-tag]
** [http://microformats.org/wiki/xfn XFN]
* used [http://microformats.org/wiki/microformats-2 microformats version 2]:
** [http://microformats.org/wiki/h-feed h-feed]/[http://microformats.org/wiki/h-entry h-entry]
** [http://microformats.org/wiki/h-card h-card]
** [http://microformats.org/wiki/comment-brainstorming#microformats2_h-feed_p-comments Comment Draft]
* used [http://www.whatwg.org/specs/web-apps/current-work/multipage/microdata.html microdata]:
** http://schema.org/Blog
** http://schema.org/BlogPosting
** http://schema.org/Comment
** http://schema.org/WebPage
** http://schema.org/Person

Planned formats:

* micormats (v2): hAudio and hMedia
* microdata: http://schema.org/MediaObject

=== WordPress Features ===

Autonomie supports:

* [http://codex.wordpress.org/Post_Formats Custom Post Formats]: aside, status, gallery, video, audio, chat, quote, link and image
* [http://codex.wordpress.org/Post_Thumbnails Post-Thumbnails]
* [http://codex.wordpress.org/Function_Reference/add_editor_style Editor-Style]
* [http://codex.wordpress.org/Navigation_Menus Navigation Menus]
* [http://codex.wordpress.org/Automatic_Feed_Links Automatic Feed Links]
* [http://codex.wordpress.org/Custom_Headers Custom-Header]
* [http://codex.wordpress.org/Custom_Backgrounds Custom-Backgrounds]
* [https://wordpress.org/gutenberg/ Gutenberg support]
** [https://wordpress.org/gutenberg/handbook/extensibility/theme-support/ Editor Colors]
** [https://wordpress.org/gutenberg/handbook/extensibility/theme-support/#wide-alignment Wide Alignment]

=== Supported Plugins ===

==== Post Kinds ====

Full support of the Post Kinds plugin:

* Post Format support
* reply-context outside of `e-content`

==== Syndication Links ====

Full support of the Syndication Links plugin:

* Syndication Links in the Post-Meta area
