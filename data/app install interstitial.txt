'''<dfn>app install interstitial</dfn>''' is an interstitial shown on a website that prompts the user to download and install an [[app]]; showing such interstitials tends to reduce user engagement with the website[http://googlewebmastercentral.blogspot.com/2015/07/google-case-study-on-app-download-interstitials.html] and is an [[anti-pattern]].
