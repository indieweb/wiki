'''<dfn>IndieWeb Utils</dfn>''' is a library that contains a set of building blocks for creating IndieWeb applications in Python, developed by {{capjamesg}}.

The library includes features such as:

* A Post Type Discovery implementation
* An Authorship Specification implementation
* A reply context generator
* A webmention endpoint discovery function
* and more...

The library is available on GitHub at [https://github.com/capjamesg/indieweb-utils https://github.com/capjamesg/indieweb-utils].

== IndieWeb Uses ==

* {{capjamesg}} is moving IndieWeb Search, his webmention endpoint, and other projects he maintains to use IndieWeb Utils where appropriate.
