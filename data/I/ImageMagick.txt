'''<dfn>[https://www.imagemagick.org/ ImageMagick]</dfn>''' is open source software for editing images, typically on the command line. For example <code>magick images/my-photo.png images/my-photo.avif</code> would create an [[avif]] version of a [[png]] file in the images/ directory. For more details on usage see [https://imagemagick.org/script/command-line-processing.php Command-line Processing].
