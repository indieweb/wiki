'''<dfn>Indigenous</dfn>''' is a set of native apps for iOS and Android currently in beta that supports posting your website using [[Micropub]] and a built-in [[reader]] that supports [[Microsub]].

== Platforms ==
The iOS [[project]] is under active development by {{eddiehinkle.com}} and can be [https://indieweb.org/indigenous_for_ios found here].

The android version is being developed by [https://indieweb.org/User:Realize.be Kristof de Jaeger] and can be [https://indieweb.org/indigenous_for_android found here].

== How to download ==
The iOS app is currently in beta testing. If you would like to help test Indigenous, you can [http://eepurl.com/c-SlfP fill out the form here].

The Android app is still in early development, but will have a link here when it is ready for beta testing.

== Feedback ==
If you have ideas/know of bugs, feel free to open up issues [https://github.com/EdwardHinkle/indigenous-ios/issues here for iOS] or [https://github.com/swentel/indigenous-android/issues here for Android]
