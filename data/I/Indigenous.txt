'''<dfn>Indigenous</dfn>''' is a set of native apps for iOS, Android currently in beta that supports posting your website using [[Micropub]] and a built-in [[reader]] that supports [[Microsub]]. A cross-platform application is also available.

== Platforms ==

==== Android ====
The android version is being developed by {{realize.be}} and can be [https://indieweb.org/indigenous_for_android found here].

<img src="https://realize.be/indigenous-screenshots/indigenous_channel_list.jpg" width="300">

==== Desktop ====

The Desktop version is being developed by {{realize.be}} and can be found at https://github.com/swentel/indigenous-desktop/ A video with features of the first release is available at https://www.youtube.com/watch?v=7egdRBg70XA

==== iOS ====
The iOS [[project]] is under active development by {{eddiehinkle.com}} and can be [https://indieweb.org/Indigenous_for_iOS found here].

<img src="https://indieweb.org/images/2/2a/indigenous-0.3-reader.png" width="300">

== How to download ==
The iOS app is currently available for Pre-Order in the iOS App Store (for free). You can [https://itunes.apple.com/us/app/indigenous-social-timeline/id1271004584?mt=8 pre-order it here]

The Android app is available on [https://play.google.com/store/apps/details?id=com.indieweb.indigenous Google Play] and [https://f-droid.org/en/packages/com.indieweb.indigenous/ F-Droid]

The desktop app is available at https://github.com/swentel/indigenous-desktop/releases for Windows, macOS and Linux.
