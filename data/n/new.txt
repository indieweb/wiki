'''<dfn>new</dfn>''' is a page for creating a new resource like a [[post]] or a [[page]] on your personal website, and also a page that links to URLs to create new resources.

== IndieWeb Examples ==

* {{capjamesg}} has a /new page as of 2026-04-19 which lists links to the URLs that let him create a new page and post on his website, as well as private documents with Proton Drive.
** The Proton Drive link is the generic https://docs.proton.me/new link, so anyone clicking it would see a document in their own drive. The UX flow for James is to go to /new, click the "Create new document" link, then copy the actual document link to share with friends.

== URLs to create new web resources ==

The URLs below, when visited in the relevant software and when signed in, will let you create a new resource like a page or a post.

=== Ghost ===

* Create a new page with Ghost: https://example.com/ghost/#/editor/page/
* Create a new post with Ghost: https://example.com/ghost/#/editor/post/

== Other examples ==

* The Google suite of web tools offers several .new URLs that let you create documents, like:
** https://docs.new
** https://sheets.new
** https://slides.new
