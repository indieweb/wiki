'''<dfn>ncdu</dfn>''' is a unix tool providing an interactive command line interface for showing disk usage.

[[File:ncdu-screenshot.png|600px]]

ncdu shows a list of folders and their sizes with a visual indicator. You can navigate into a folder to see the contents and sizes of its subfolders. This can be used to quickly find which folders are using a disproportionate amount of disk space.
