The '''<dfn>finger protocol</dfn>''' is a protocol used to transmit user information as plain text. While the finger protocol runs independently of [[http]], there is a similar web-based protocol for discovery and identification called [[WebFinger]].

Finger is still used on some<ref>The [https://tildeverse.org tildeverse] and sites belonging to it support [https://portal.mozz.us/finger/tildeverse.org/ finger]</ref><ref>[http://SDF.org SDF] support [https://portal.mozz.us/finger/sdf.org/ finger]</ref> community [[pubnix]]es, where many users on the shared server can identify themselves internally and publically.

To request information over finger, usually the [[TUI]] is employed, via e.g. <code>finger</code> command or terminal based browsers with finger support such as [https://en.wikipedia.org/wiki/Lynx%20(web%20browser) Lynx] &mdash; however the graphical [[Gemini protocol|Gemini]] client [https://gmi.skyjake.fi/lagrange/ Lagrange] also supports finger addresses via <code>finger://</code> [[uri|URI]] scheme. Additionally some web [[proxy|proxies]] to the finger protocol exist such as the [https://portal.mozz.us/ Smallnet Portal]
