'''<dfn>first name</dfn>''' is an archaic Western-centric term for a person’s given name that is still used in numerous coding examples and even supposed ”[https://chat.indieweb.org/microformats/2022-01-30#t1643503776668700 protocols]”; please avoid using any notion or label of “first name” in code or UIs, and instead use “given name” which has been in [[vCard]] since the 1990s.
