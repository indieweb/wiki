'''<dfn>[https://wallabag.org Wallabag]</dfn>''' is a free-software tool for saving/bookmarking web pages, in the style of Instapaper and Pocket. This [https://framabag.org/cquoi/ animated comics] is a presentation of the philosophy and use-cases of Wallabag.

== Features ==

* Import bookmarks from Pocket and Instapaper
* Install anywhere (see [[web hosting]])
* Responsive design (fits mobile needs)
* RSS feeds

== Public instances ==

You can use Wallabag at one of the following public instances for free-price :
* [https://framabag.org Framabag], hosted by [https://framasoft.org Framasoft] association ([https://soutenir.framasoft.org/?f=nav donation])

If you know of more public instances, please add them to the list.
