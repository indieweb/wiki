'''<dfn>[https://wallabag.org Wallabag]</dfn>''' is a free-software tool for saving/[[bookmark|bookmarking]] web pages, in the style of Instapaper and [[Pocket]]. This [https://framabag.org/cquoi/ animated comics] is a presentation of the philosophy and use-cases of Wallabag.

== Features ==

* Import bookmarks from Pocket and [[Instapaper]]
* Install anywhere (see [[web hosting]])
* Responsive design (fits mobile needs)
* [[RSS]] feeds
* Mobile apps
* Browser extensions

== Public instances ==

You can use Wallabag at one of the following public instances:

* [https://www.wallabag.it/ Wallabag It] is managed by the project itself and charges a small hosting fee.
If you know of more public instances, please add them to the list.
