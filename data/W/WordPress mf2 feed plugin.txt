The Wordpress '''<dfn>[https://wordpress.org/plugins/mf2-feed/ MF2 Feeds]</dfn>''' plugin provides a Microformats2 JSON and [[JF2]] representation for every WordPress URL, to allow other sites to get microformats data even if the theme used does not support Microformats2. The source code is on [https://github.com/indieweb/wordpress-mf2-feed Github]
