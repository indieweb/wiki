<dfn>'''[https://wrimini.net/ Wrimini]'''</dfn> is a [[Micropub/Clients|Micropub client]] for Android. It supports notes, replies, likes, bookmarks and photo uploads to a media-endpoint.

There is a [https://play.google.com/store/apps/details?id=eu.stuifzand.micropub client release on the Play Store] as an Alpha release. You need to be added to the list of testers, to be able to download it there. Contact Peter Stuifzand, if you like to be added.


=== Supports ===
* Notes
* Articles
* Replies
* Likes
* Bookmarks
* Media endpoint
* Post status
* Visibility


The source code can be found on [https://github.com/pstuifzand/micropub-android Github].
