<dfn>'''[https://github.com/pstuifzand/micropub-android Wrimini]'''</dfn> is a Micropub client for Android. It supports notes, replies, likes, bookmarks and photo uploads to a media-endpoint.

There is a client release on the Play Store as an Alpha release. You need to be added to the list of testers, to be able to download it there. Contact Peter Stuifzand, if you like to be added.


* The source code can be found on [https://github.com/pstuifzand/micropub-android Github].
