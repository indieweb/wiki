'''<dfn>[https://developer.mozilla.org/en-US/Add-ons/WebExtensions WebExtensions]</dfn>''' are an API developed by the [https://www.w3.org/community/webextensions/ WebExtensions Community Group] for building [[browser add-on]]s.

API draft: https://github.com/w3c/webextensions/blob/main/index.bs

== Browser Documentation ==
As there isn’t a single standard (yet), [[browsers]] publish their own documentation. 

[[MDN]] publishes an overview:
* [https://developer.mozilla.org/en-US/Add-ons/WebExtensions/Browser_support_for_JavaScript_APIs browser support of the separate APIs].

Browser documentation:
* [[Google Chrome]]: [https://developer.chrome.com/extensions Extensions]
* [[Microsoft Edge]]: [https://docs.microsoft.com/en-us/microsoft-edge/extensions Microsoft Edge extensions]
* [[Mozilla Firefox]]: [https://developer.mozilla.org/en-US/Add-ons/WebExtensions Browser Extensions]
** [https://mozilla.github.io/extension-finder/ Find a Compatible Alternative to your Firefox Legacy Extension]
* [[Opera]]: [https://dev.opera.com/extensions/ Extensions Documentation]
* [[Safari]]: [https://developer.apple.com/documentation/safariservices/safari_web_extensions Safari Web Extensions]
