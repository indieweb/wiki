'''<dfn>XRay</dfn>''' is an open source API that returns structured data for a URL by parsing [[microformats]] and following other indieweb [[algorithms]], and is part of the [[p3k]] suite of applications.

The instance of XRay run by {{aaronpk}} at https://xray.p3k.io is used by [[webmention.io]] to parse comment text, and used by [[Loqi]] to display a text summary of URLs in the [[IRC]] channel.

You can try the API with your own URL or by pasting in HTML in the form at this link: 
* https://xray.p3k.io
