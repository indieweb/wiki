'''<dfn>[https://github.com/indieweb/bsky.link bskylink]</dfn>''' is an [[open source]] link wrapper that generates a simple, embeddable version of a post on [[Bluesky]], marked up with [[microformats2]] data, and can generate links to threads and mf2 feeds for user profiles. It was previously hosted at <nowiki>https://bsky.link</nowiki>.

== Syntax ==

=== Generate link to a single post ===

<pre>
/?url=https://staging.bsky.app/profile/jamesg.blog/post/3jttk3dwqkk2e
</pre>

=== Generate link to a thread ===

<pre>
/?url=https://staging.bsky.app/profile/jamesg.blog/post/3jv2hn4dcsu2f&show_thread=t
</pre>

=== Generate an mf2 feed for a user profile ===

<pre>
/feed?user=jamesg.blog
</pre>

Where <code>jamesg.blog</code> is the user's handle on Bluesky.
