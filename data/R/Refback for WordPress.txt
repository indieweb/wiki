'''<dfn>[https://wordpress.org/plugins/refback/ Refback]</dfn>''' is a plugin for [[WordPress]] which enables [[refbacks]] functionality on a WordPress site and has been abandoned by its former users.

== Needs updating ==
The plugin is in need of update as it requires [[Semantic Linkbacks]] for progressive enhancement and that plugin was retired/merged into Webmentions.

== Versions ==
* Official WordPress Repository version: https://wordpress.org/plugins/refback/
* Development version on GitHub: https://github.com/dshanske/wordpress-refback/

== IndieWeb Examples ==
No current IndieWeb Examples. Everyone in the community that used to use this plugin has disabled or uninstalled it.

=== Previous Examples ===
* {{GWG}} developed the plugin but is no longer using it due need for update.
** He also maintains many of the other [[linkback]] related plugins in the WordPress ecosystem
* {{chrisaldrich}} used this plugin on [https://boffosocko.com boffosocko.com] from its inception until 2023-05-30 when Semantic Linkbacks was merged into the Webmention plugin.
