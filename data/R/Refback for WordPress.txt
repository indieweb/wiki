'''<dfn>[https://wordpress.org/plugins/refback/ Refback]</dfn>''' is a plugin for [[WordPress]] which enables [[refbacks]] functionality on a WordPress site.

It is in need of update as it requires [[Semantic Linkbacks]] for progressive enhancement and that plugin was retired/merged into Webmentions.

== Versions ==
* Official WordPress Repository version: https://wordpress.org/plugins/refback/
* Development version on GitHub: https://github.com/dshanske/wordpress-refback/
