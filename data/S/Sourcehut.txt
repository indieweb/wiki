'''<dfn>Sourcehut</dfn>''' (AKA <dfn>sr.ht</dfn>) is an open-source platform for hosting code, version control, and a build automation system.

== IndieWeb Examples ==

=== capjamesg ===

{{capjamesg}} uses chat.sr.ht, the IRC bouncer and web interface developed by Sourcehut, for interacting with IRC communities.

capjamesg formerly mirrored his Git repositories from GitHub to sr.ht using a custom script.
