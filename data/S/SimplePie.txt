'''<dfn>[http://simplepie.org/ SimplePie]</dfn>''' is a [[feed]] parser library written in PHP, which supports [[h-feed]] parsing as of version 1.4. The latest version is available on GitHub: https://github.com/simplepie/simplepie

== History ==
* Version 1.4 was released 2016-04-25
* Version 1.4-beta was released on 2016-04-10 and includes support for parsing [[h-feed]]s.
