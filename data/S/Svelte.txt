'''<dfn>[https://svelte.dev/ Svelte]</dfn>''' is a [[JavaScript]] framework for making interactive web pages, can be turned into a [[static site generator]], and has been used by some community members.

The [https://svelte.dev/docs/kit/introduction#What-is-SvelteKit Sveltekit] framework can turn Svelte into a static site generator.

== IndieWeb Examples ==
=== Jacky Alciné ===
* {{jacky}} uses Svelte for building the client-facing aspects of the Web version of [[Sele]], [[Shock]] and [[Lighthouse]]. {{jacky}} also plans to use it for building his personal site's theme to make it more interactive.
