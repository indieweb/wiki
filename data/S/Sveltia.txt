'''[https://sveltiacms.app/en/ Sveltiacms]''' is a Git-based headless [[CMS]]. It works well with any static site generator, such as [[hugo]], [[11ty]], and [[Astro]].

This kind of CMS is useful if you enjoy the benefits of [[static site generators]] but still want to quickly add or edit content from mobile devices or other situations where using Git directly is cumbersome. Unlike most traditional CMS, Sveltia is entirely on the client side, so there is no server and database to host and manage, and your edits are written directly back to the Git repository.
