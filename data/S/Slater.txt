'''<dfn>Slater</dfn>''' is an app for posting events to your own website using [[Micropub]]. It is created by [[User:Martymcgui.re|Marty McGuire]].

Slater is built on PHP and the code is open source: https://git.schmarty.net/schmarty/slater-php/

== Features ==

IndieWeb features that Slater supports:

* publishing [[event]] (<code>h: event</code>) posts with:
** name
** start date + time
** end date + time
** timezone via offset
** location as an opaque string (suggested either [[geo]] or a url with a [[venue]] [[h-card]]
** [[tags]]
** [[featured]] image (supports sending multiple if your micropub config reports a media endpoint).
** description
** [[mp-syndicate-to]]
