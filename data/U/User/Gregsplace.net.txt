== Server Environment ==

[https://gregsplace.net Greg's Place] is hosted on a 4 GB Dedicated instance in [https://linode.com Linode's] Frankfurt, Germany data center and has the following specifications:

* 2 CPUs
* 4 GB RAM
* 80 GB Storage
* Dual-stack IPv4 & IPv6

The server uses the following software components to provide its current functionality:

* Ubuntu 22.04.2
* Apache 2.4.5.7
* MariaDB 10.8.8
* PHP 7.4.33
* Wordpress 6.2.1
