== Plumbing ==

* Hardware: Dedicated 4GB Linode with 2 CPU cores, 80GB storage & 4GB RAM
* OS: Ubuntu 22.04.1
* Webserver: Apache 2.4.5
* PHP Version: 8.1.2
* Database: MariaDB 10.8
* CMS: Wordpress 6.1.1

== Following & Providing Feedback ==

* Links to content are posted to [[https://twitter.com |Twitter]] where simply replying to the tweet containing the content you're interested in will forward your response via the [[https://indieweb.org/webmention |Webmention]] protocol back to the website and post it as a comment underneath the content that the tweet referenced.
* You can follow the website using your preferred [[https://joinmastodon.org/ |Mastodon]] instance by copying & pasting the following URL into the search box provided on the instance's page or in your preferred app: https://gregsplace.net/author/greg/ and clicking or tapping the search button provided. A follow button can be found within the search results, which can be clicked or tapped on and you will begin receiving content from the website in your [[https://joinmastodon.org |Mastodon]] account. Once the previous steps are complete, simply replying to the toot containing the content you're interested in will forward your response via the [[https://indieweb.org/ActivityPub |ActivityPub]] protocol back to the website and post it as a comment underneath the content that the toot referenced.
