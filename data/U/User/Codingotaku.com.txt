== About ==

[https://codingotaku.com Coding Otaku] is a Generalist software developer from India, currently living in London, United Kingdom.

To track the Indie Web progress, Coding Otaku has a dedicated checklist page named [https://codingotaku.com/am-i-indieweb-yet Am I IndieWeb Yet?]

The all currently implemented IndieWeb features for [https://codingotaku.com codingotaku.com] except [[IndieLogin]] are done in-house with Rust, and the source code can be found at [https://codeberg.org/codingotaku/website codeberg.org/codingotaku/website].
