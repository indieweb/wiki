== English version==
=== About Senzaki Yakumo ===
I am Senzaki Yakumo. I run a personal blog called <a href="https://lastencore.org/" hreflang="ja" rel="me">Last Encore</a>, where I post daily thoughts, poetry, and various miscellaneous notes.

=== Why I built my own site ===
<p>I wanted a space where I could write my thoughts calmly and thoughtfully, without getting caught up in today's internet culture of "flaming," going viral, or the constant pressure to make an impact in mere seconds.</p>

<p>To that end, I tried <a href="https://wordpress.com/">WordPress</a>, but I couldn't get it to meet my standards for source code cleanliness, accessibility, and <abbr title="JavaScript Object Notation for Linked Data">JSON-LD</abbr> support.</p>
<p>Seriously, why is it so impossible to eliminate all those unnecessary <code class="language-html">div</code> tags in WordPress block themes? On the other hand, falling back to a classic or hybrid theme felt like a frustrating admission that I couldn't keep up with the times.</p>
<p>Just as I was despairing and ready to give up and return to old-fashioned hand-coding, I discovered the static site generator <a href="https://astro.build/">Astro</a>, which finally allowed me to build a blog I am truly satisfied with. I am so grateful for Astro.</p>
<p>As a side note, returning to <abbr title="HyperText Markup Language">HTML</abbr> and <abbr title="Cascading Style Sheets">CSS</abbr> for the first time in decades was a revelation. I was moved to find that we no longer have to struggle with <code class="language-css">float</code> for layouts, rounded corners can be implemented with pure CSS instead of images, and the curse of the dreaded Internet Explorer has finally been lifted.</p>

=== Itches ===
* Maximize accessibility to create a site that everyone can enjoy.

== 日本語版 ==
=== センザキヤクモについて ===
<p>わたしは、センザキヤクモといいます。<a href="https://lastencore.org/" rel="me">Last Encore</a>という、日々思ったことや詩などの雑記系ブログをやってます。</p>

=== 自分のサイトをつくった理由 ===
<p>炎上やバズり、短い時間でいかにインパクトを与えるか。そういう昨今のネットのありかたに巻き込まれず、淡々と自分の思うことを、それなりの文章で書きたかったからです。</p>
<p>というわけで、<a href="https://wordpress.com/ja/">WordPress</a>を試しましたが、ページのソースコードやアクセシビリティ、および<abbr title="JavaScript Object Notation for Linked Data">JSON-LD</abbr>への対応など、納得いくものが作れませんでした。

本当に、どうしてWordPressのブロックテーマは<code class="language-html">div</code>を消し去れないのでしょうね。かといって、クラシックテーマとかハイブリットテーマにするのも、時代についていけなくて悔しい。</p>
<p>もう昔ながらの手打ちでいいかと絶望していたところで、静的サイトジェネレーターの<a href="https://astro.build/" hreflang="en">Astro</a>にたどり着き、自分の満足いくブログサイトをつくることができました。Astroに感謝。</p>
<p>余談ですが、数十年ぶりに<abbr title="HyperText Markup Language">HTML</abbr>や<abbr title="Cascading Style Sheets">CSS</abbr>に触ったところ、<code class="language-css">float</code>での位置調整に悩まされなくなっていたり、角丸が画像ではなくCSSだけで実装できるようになっていたり、あの憎きInternet Explorerの呪いが浄化されていて、感動しました。</p>
