== my indieweb setup  ==

* https://matienzo.org/ is running [[Hugo]], with support for [[Webmention]] (via [[Webmention.io]])
** I post outgoing webmentions manually (usually through CLI or [[Telegraph]])
** I'm using a gently forked version of {{beesbuzz.biz}}'s [[webmention.js]] to display incoming webmentions
** I use [[Bridgy]] for [[backfeed]] and [[POSSE]] 
* I use [[Netlify CMS]] for editing notes while I think through how I want to implement Micropub.
