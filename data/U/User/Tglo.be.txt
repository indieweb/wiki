== Current setup ==
A self build static site generator called Nordheim (after the [https://en.wikipedia.org/wiki/Arne_Nordheim composer]. The webpages are living on [Github Pages]. Currently only supporting notes, replies, articles and likes.

===Tools used===
* Python
* Markdown
* webmention.io for revieving webmentions
* Zoho Mail
* Github Pages
* indieauth.com
