== Site details ==

* Original site content written in <a href="https://orgmode.org/">Org mode</a> (<a href="https://www.gnu.org/software/emacs/">GNU Emacs</a>). That content is then exported to Markdown using <a href="https://ox-hugo.scripter.co"><code>ox-hugo</code></a>.
* Built using [[hugo|Hugo]].
* Deployed using <a href="https://netlify.com">Netlify</a>.
* The site's HTML, CSS, JS is minified using a Go commandline utility <a href="https://github.com/tdewolff/minify"><code>minify</code></a>.
* Uses [[webmention.io]] as Webmention endpoint.
* Supports receiving comments via Webmentions (via URL or comment using [[commentpara.de]]), or as back-feed from [[twitter]] if a post is syndicated there. The back-feed Webmentions are generated by [[brid.gy]].
* Pages are marked with <code>h-feed</code>, <code>h-entry</code>, and the nested <code>p-</code> and <code>u-</code> as needed. Please ping me on IRC if you find something incorrect or unusual about my microformats2 markup.
* <code>h-card</code> is added to the footer of each page.
