A '''<dfn>trip</dfn>''' is a post that describes the act of traveling from a start location to an end location by some mode of transport.

== How To ==

A trip may include the following components

* start location
* end location
* start time
* end time
* route
* mode of transport
* cost
* carbon emissions (using a tool like [http://www.ecopassenger.org/ ecopassenger.org])
* total distance
* total duration

A trip may contain more than one [[leg]].
