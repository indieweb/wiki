'''<dfn>kaku</dfn>''' is an open-source Python [[Flask]] app that can be used to handle incoming [[micropub]], [[webmention]] and [[token]] requests. 

* open source: https://github.com/bear/kaku
* self-dogfooded live on https://bear.im/
* generates static HTML from markdown source files
* Supports dt-updated, dt-published and dt-deleted
* Currently passing all of the Webmention.rocks tests
* Micropub endpoint handles create, delete and undelete

== Post Types ==
Currently supported creating of
* '''[[articles]]'''
