'''<dfn>Slater</dfn>''' is an app for posting events to your own website using [[Micropub]]. It is created by [[User:Martymcgui.re|Marty McGuire]].

Slater is built on the Python [[Flask]] framework and makes use of {{ kylewm }}'s [https://github.com/kylewm/flask-micropub flask-micropub] library for handling [[IndieAuth]] and finding Micropub endpoints.

== Features ==

IndieWeb features that Slater supports:

* publishing [[event]] (<code>h: event</code>) posts with:
** name
** start date + time
** end date + time
** timezone via offset
** location as a url (suggested either [[geo]] or a url with a [[venue]] [[h-card]]
** [[tags]]
** [[photo]] (supports sending multiple if your micropub config reports a media endpoint). useful for [[featured]] image.
** description
** [[mp-syndicate-to]]
