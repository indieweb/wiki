'''<dfn>[http://owncloud.org ownCloud]</dfn>''' is an open source software [[project]] for self-hosted personal web services including a file manager (an indie version of [[Dropbox]]), music, calendar (CalDAV), contacts (CardDAV), and more.

== Applications ==
* [https://github.com/owncloud/news ownCloud News] - an RSS/Atom feed aggregator. It offers a RESTful API for app developers.


== Forked to Nextcloud ==
In early June 2016, ownCloud was forked to create [https://nextcloud.com/ Nextcloud]. [https://youtu.be/iMfokaX2r8g Discussion and reasons for the fork].
