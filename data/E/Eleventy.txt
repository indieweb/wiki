'''<dfn>[https://www.11ty.io/ Eleventy]</dfn>''' is a [[JavaScript]] based [[static site generator]] with a decoupled architecture intended to allow the user to select their own preferred template engines. Eleventy is not a JavaScript framework and does not come with a preinstalled theme or configuration although there are many examples of [https://www.11ty.io/docs/starter/#starter-projects starter projects]. 

Like other static site generators the learning curve for Eleventy can be quite steep without prior experience of writing HTML.

* GitHub: https://github.com/11ty/eleventy/
