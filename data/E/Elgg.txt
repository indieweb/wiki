'''<dfn>[https://elgg.org/ Elgg]</dfn>''' is an open source social networking engine that is IndieWeb [[friendly]]. IndieWeb features can be added with the [https://github.com/RiverVanRain/indieweb indieweb plugin].

The Elgg project was started in 2004 by {{benwerd}} and [https://twitter.com/davetosh Dave Tosh]. It focuses on educational environments, but can be used by any organization/group.

== IndieWeb Friendly ==
2024-10-18 [https://elgg.org/plugins/3292492 IndeWeb integration for Elgg v2.0.2] plugin released (for Elgg v6)!

2023-12-11: [https://elgg.org/plugins/3292492 IndieWeb integration for Elgg v1.1.0] plugin released!
* [[Webmention]]: send and receive; send to syndicate content, likes etc via [[brid.gy]]
** store [[syndication link]]s
** auto-create comments from [[in-reply-to]]
** [[reply]] to comments and send webmentions via Webmention.io and other external or internal endpoint
* [[microformats]]: apply [[microformats2]] to your markup for content, images, and more
* [[IndieAuth]]: allow users to login and create accounts with IndieAuth; built-in IndieAuth Authorization and Authentication API; expose endpoints and use external or internal endpoint
* [[Micropub]]: Create, update, and delete content [[note]], [[article]], [[event]], [[rsvp]], [[reply]], [[like]], [[repost]], [[bookmark]], [[checkin]], and [[issue]]
* [[Microsub]]: built-in server or use external service; expose a Microsub endpoint, external or internal
* [[feed file|Feeds]]: create [[JF2]] feeds
* Media cache: store images locally for internal webmention and microsub endpoint
* [[WebSub]]: PuSH 0.4 for publishing and subscribing, integrates with Microsub support for feed subscription
* [[contacts|Contacts]]: store contacts for Micropub contact query, allowing for autocomplete
