'''<dfn>Elixir</dfn>''' is an open source functional programming [[language]] that runs on the [[Erlang]] virtual machine, with software and libraries for personal sites and IndieWeb [[building blocks]].

== IndieWeb Examples ==
* https://wwwtech.de (by [[User:Wwwtech.de_about]]) is written in Elixir with [[Phoenix]]: https://github.com/ckruse/wwwtech.de
* [[Koype]] is being written in Elixir with [[Phoenix]]

== Libraries ==

* [[microformats2]] parser: http://microformats.org/wiki/microformats2#Elixir https://github.com/ckruse/microformats2-elixir
* [[webmention]] libraries:
** https://github.com/tjgillies/webmention
** https://github.com/ckruse/webmentions-elixir
* [[micropub]] library: https://github.com/bismark/plug_micropub
* HTTP Link header library: https://github.com/myfreeweb/ex_http_link
