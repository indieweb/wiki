'''<dfn>[https://github.com/pstuifzand/ekstereye Ekster Reader]</dfn>''' is a [[Microsub]] reader created by {{pstuifzand}} and runs in a web browser. It connects directly to a Microsub server without its own Microsub proxy backend. For this to work you do need a Microsub server that handles [[CORS]] headers, to allow access from the browser. This also applies for the [[IndieAuth]] endpoint. A version is hosted at https://reader.p83.nl/

== Features ==
* IndieAuth
* Feed search
* Preview of feed
* Item search (when supported by server)
* Like, reply, repost, bookmark from an item
* Follow a feed from an item
