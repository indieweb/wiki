'''<dfn>[https://bookwyrm.social/ BookWyrm]</dfn>''' is a social book reading, reviewing, and [[personal library]] site built on open source software that supports [[ActivityPub]]. 

The source code is available:
* https://github.com/bookwyrm-social/bookwyrm

== History ==
The project was previously known as [https://github.com/mouse-reeve/fedireads FediReads] - an [[ActivityPub]] based [[Goodreads]] like service with https://openlibrary.org/; [https://fedireads-test.glitch.me/ Demo previously available on Glitch]
* 2020-02-19 Announcement thread: [https://web.archive.org/web/20200315235453/https://twitter.com/tripofmice/status/1230235138341998592 https://twitter.com/tripofmice/status/1230235138341998592] <blockquote><p>Are you interested in a federated alternative to Goodreads that doesn't use Amazon?</p><p>because I'm making a federated alternative to Goodreads that doesn't use Amazon</p></blockquote>
