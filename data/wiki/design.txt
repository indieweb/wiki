== CSS ==

=== Custom Page CSS ===
You can add custom CSS to a page by using the <nowiki><raw></nowiki> tag. For example, see the [https://indieweb.org/wiki/index.php?title=Main_Page&action=edit CSS on the home page].

=== Global CSS ===
MediaWiki provides a global CSS file that can be edited like a wiki page. 

* [[MediaWiki:Common.css]]

=== User Styles ===
When you're logged in, you can edit a stylesheet that applies only to yourself. You can find the link under [[Special:Preferences]], it will be a page named like [[User:Aaronparecki.com/common.css]].

== Resources ==
The resources below were useful in choosing colors and design of this wiki.

* http://leaverou.github.io/contrast-ratio/ - Check the contrast ratio of two colors. Useful when deciding what color text to put onto a colored background.
* http://webaim.org/resources/contrastchecker/ - another contrast ratio checker; WebAIM has additional accessibility tools as well.
* https://github.com/okayjeffrey/Scalor - Generate many shades of a set of 5 main colors.
* http://www.0to255.com - another color variation generator
* http://paletton.com/#uid=60q0t0kFAHm+-oWT2uAm-vT8ata - another color variation generator
