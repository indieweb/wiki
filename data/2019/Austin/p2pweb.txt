== Notes ==
* Dat browser is a fork of Chrome
* Create a new keypair which gives you an address
* Showing creating a new web page in a browser
* drop-in replacement for HTTP, no need to set up a server since it's p2p between browsers
* goal is to have people running their own websites and interacting with each others websites
* showing paul frazee personal website on dat
* view source -> duplicate website
* creating a social media feed by posting JSON files
