'''<dfn>[https://js-naked-day.org/ JS Naked Day]</dfn>''' is an event on April 24th of each year where participants remove the [[JS]] on their website. It started in 2024.

== Motivation ==
<blockquote>“JS Naked Day promotes the rule of least power. That is: start with HTML for semantic markup and CSS for styling. Use the web platform (especially forms!) for interactivity. And then, progressively upgrade with JavaScript for advanced interactivity. In plain terms: your website should work without JavaScript enabled.”

<cite>https://js-naked-day.org/</cite>
</blockquote>
