'''<dfn>[https://github.com/aaronpk/TikTokBot TikTokBot]</dfn>''' is an [[IRC]] and [[Slack]] bot API written by {{aaronpk}}.

== IndieWeb Examples ==
* [[Loqi]] is built on top of TikTokBot.
* {{sknebel}} uses TikTokBot to get IRC notifications for incoming Webmentions.
* {{sknebel}}'s IRC bot [[Kaja]] is based on it.
* {{sebsel}} is using a TikTokBot named 'bop' on the [[Raspberry Pi]] that runs his IRC bouncer (and a private IRC network for it), to get notifications of Webmentions, logins and Micropub actions.
