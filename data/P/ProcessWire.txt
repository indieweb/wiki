<dfn>'''[https://processwire.com/ ProcessWire]'''</dfn> is an open source [[PHP]] content management system/content management framework.

== Plugins ==
=== ProcessWire Webmention ===
'''[https://modules.processwire.com/modules/webmention/ ProcessWire Webmention]''' ([https://github.com/gRegorLove/ProcessWire-Webmention source code]) adds webmention support to your ProcessWire site. Includes support for [[Vouch]] extension and an admin interface to view, process, delete, or change the visibility of received webmentions. It can also monitor a [[blogroll]] or [[allowlist]] and add new links it finds to the list of approved vouch domains. {{gRegor}} maintains and uses this plugin on gregorlove.com since 2015-07-12.
* v2.0.0 released 2018-05-06 [https://gregorlove.com/2018/05/webmention-for-processwire-update/]
* v1.1.3 released 2017-03-27 [https://gregorlove.com/2017/03/webmention-for-processwire-update/]
* v1.1.2 released 2016-04-11
* v1.1.1 released 2016-04-10
* v1.1.0 released 2016-02-25 in the ProcessWire Modules Directory [https://modules.processwire.com/modules/webmention/]
* Issues, pull requests: https://github.com/gRegorLove/ProcessWire-Webmention

=== ProcessWire IndieAuth ===
'''[https://processwire.com/modules/process-indie-auth/ProcessWire ProcessWire IndieAuth]''' ([https://github.com/gRegorLove/ProcessWire-IndieAuth source code]) allows you to use your domain to sign in to applications and grant access to read/write to your site using [[IndieAuth]]. {{gRegor}} maintains and uses this plugin on gregorlove.com since 2016-03-17.
