'''Pagecord''' is an open source + hosted blogging service created by [[User:Olly.world|Olly Headey]]. Posts can be written in the built-in editor, sent by email, published via [[Micropub]],
a REST API with CLI, or the [https://community.obsidian.md/plugins/pagecord official Obsidian plugin].

== IndieWeb support ==
* Native [[Micropub]] server support for hosted blogs
* [[microformats2]]: blog pages are marked up with [[h-feed]], [[h-entry]], [[h-card]], e-content, and dt-published
* [[rel-me]]: a Mastodon link in a blog's social links is output as <code>rel="me"</code> for profile verification
* [[RSS]] feeds for every blog, plus email subscriptions for readers
* Custom domain support
* Full content export (Markdown or HTML)

== Details ==
* Open source: [https://github.com/lylo/pagecord github.com/lylo/pagecord]
* Free plan (unlimited posts, no hosted images), or a paid premium option
* Launched [https://blog.pagecord.com/hello-world-5f5e111 March 2024]
