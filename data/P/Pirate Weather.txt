'''<dfn>[https://pirateweather.net Pirate Weather]</dfn>''' is a free and open weather forecast API created by Alexander Rey, and supported by the [[Simple Location]] WordPress plugin. 

== Features ==
* Forecast data, including current conditions, minutely data for one hour, hourly data for up to 168 hours, and daily data for 7 days should be available. Weather alerts are working in the US.
* drop-in replacement for [[Dark Sky]] API calls.

== Clients ==
* [[Simple Location]] WordPress plugin by {{gwg}} can use the Pirate Weather API among several weather APIs.
