'''<dfn>[https://github.com/fdv/publify/ Publify]</dfn>''' is a Ruby on Rails blogging engine [[project]] with extended publishing capacities that's recently been taking a turn as an Indie Web project. Publify and supported plugins are free software released under the MIT licence. Publify needs a database like MySQL, PostgreSQL or SQLite.

== Features ==
Master currently supports the following indie Web features:
* Classic blogging engine with API, plugins, [[RSS]]/[[Atom]]...
* [[POSSE]] to [[Twitter]] with short messages
* [[h-review]] on default themes
* Self hosted URL shortener 
* [[PESOS]] for formerly posted tweets
* Markdown, Textile, Flickr import, auto link. Twitter handler discovery and syntax highlighting.
