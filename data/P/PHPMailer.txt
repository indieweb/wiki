'''<dfn>[https://github.com/PHPMailer/PHPMailer PHPMailer]</dfn>''' is a popular library for sending email from PHP, including [[WordPress]], though renamed as class-phpmailer.php.

== Security ==
PHPMailer publishes a list of security updates at https://github.com/PHPMailer/PHPMailer/blob/master/SECURITY.md

In 2016-12, a remote code execution [[security]] vulnerability was reported [https://legalhackers.com/advisories/PHPMailer-Exploit-Remote-Code-Exec-CVE-2016-10033-Vuln.html][https://twitter.com/hanno/status/813309977121222656]. According to the security updates linked above, it was fixed in version 5.2.18.
