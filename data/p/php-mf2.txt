'''<dfn>[https://github.com/indieweb/php-mf2 php-mf2]</dfn>''' is a superb open source PHP [[microformats2]] parser that also supports classic [[microformats]] per the microformats2 parsing specification.

== Composer ==
php-mf2 has PHP [[Composer]] support and that is the recommended way to [https://github.com/indieweb/php-mf2#installation install and use it].

== Just one file ==
You can also use php-mf2 with the "just use this one file" approach by following the instructions at https://github.com/indieweb/php-mf2#installation
<blockquote>You could install it by just downloading [https://raw.github.com/indieweb/php-mf2/master/Mf2/Parser.php /Mf2/Parser.php] and including that</blockquote>
