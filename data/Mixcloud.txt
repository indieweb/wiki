'''Mixcloud''' is a silo for hosting DJ sets/mixtapes and other mixed audio. It's used by DJs, radio shows, nightclub promoters and so on.

There is potential to use their API for [[PESOS]]: [http://api.mixcloud.com/tom-morris5/feed/ example feed] (from [[User:Tommorris.org|tommorris]]), [https://www.mixcloud.com/developers/ API documentation].
