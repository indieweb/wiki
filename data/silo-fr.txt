= Silo =
Un <dfn>silo</dfn>, ou silo d'hébergement de contenu web, dans le contexte de l'IndieWeb, est un site web qui : 
* vous oblige à créer un compte spécifique à ce site pour l'utiliser (silo d'identité)
* vous permet d'interagir sur le site <em>uniquement</em> avec d'autres comptes sur le site (silo de contacts / réseau social)
* vous permet de poster certains types de contenus (texte, hypertexte, images, vidéos)
