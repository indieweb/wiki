'''<dfn>[https://github.com/EdwardHinkle/abode abode]</dfn>''' is {{eddiehinkle.com}}'s open source IndieWeb personal website software, in use at [http://eddiehinkle.com eddiehinkle.com] since 2017-02-01. All [https://github.com/EdwardHinkle/abode source code] is available on GitHub. Content and media are currently also stored in the same git repository, however in future versions this is going to be separated.

== Usage ==
Currently being used to run the following sites:
* [https://eddiehinkle.com eddiehinkle.com]
