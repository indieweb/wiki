= Facebook =
Facebook.com est un [[silo-fr|silo]] populaire d'hébergement de contenus plus souvent connu comme étant le plus grand réseau social centralisé sur le web.

== Exporter vos données ==
Vous pouvez exporter vos data sur Facebook en :
* allant sur : https://www.facebook.com/settings
* cliquant sur le lien "Télécharger une copie" en bas du texte.

== POSSEr vers ==
Il existe de nombreux moyens de [[POSSE-fr|POSSE]]r à partir de votre site indieweb vers votre profil Facebook.

Les techniques existantes :

=== à partir de Twitter ===
* [[User:Tantek.com|Tantek]] POSSE à partir de tantek.com vers Twitter, et puis utilise la fonctionnalité d'intégration Twitter/FB pour faire couler ce POSSE de Twitter vers Facebook. Raisons :
** utilité marginale insuffisante à écrire un syndicateur POSSE personnalisé vers FB.
** FB ne reçoit que des résumés abrégés (par conséquent, met plus de valeur sur les posts originaux tantek.com).
* [[User:Tommorris.org|tommorris]] fait pareil, parce qu'il est paresseux.

=== IFTTT ===
* [[User:Adactio.com|Adactio]] POSSE à partir de adactio.com vers Facebook en utilisant If This Then That (IFTTT).

=== Facebook API ===
Aucun site indieweb connu ne s'est ennuyé à appeler directement l'API FB pour POSSEr vers FB.

Poster vers le mur de quelqu'un est supporté par la 'Graph API' de Facebook

Les bibliothèques de langage de programmation pour faire ainsi : 

* Ruby : [https://github.com/arsduo/koala/wiki/Graph-API Koala: Graph API] - voir la méthode <code>put_wall_post</code>.
* ...
