'''<dfn>Nimbus</dfn>''' was a company that provided a browser extension that allowed for screen capture and screen recording.
<BR>
<BR>
They have since been sold and rebranded, now focusing on providing AI-integrated workspaces for businesses.  

== Aquired ==

On August 31, 2023, MYOB announced it acquired Nimbus Portal Solutions. https://www.myob.com/au/press-releases/myob-keeps-head-in-the-cloud-with-acquisition-of-nimbus

== Rebranded ==

On October 6, 2023 Nimbus was rebranded as FuseBase. https://medium.com/@fusebase/the-next-stage-of-client-collaboration-nimbus-evolves-into-fusebase-2db6354bea7a
<BR>
<BR>
Fusebase now provides business-to-business workspaces with AI integration that focus on collaboration, client management, and workflows.  

== Hacked ==
In February 2025, several Chrome extensions, including the Nimble Capture web clipper tool by Nimbus, were hacked and found to contain malware. Subsequently, the extension was removed from the Chrome Web Store. https://www.tomsguide.com/computing/online-security/3-2-million-chrome-users-at-risk-from-malicious-extensions-delete-them-right-now
<BR>
<BR>
There's a Reddit discussion thread about it here. https://www.reddit.com/r/chrome_extensions/comments/1iklr8q/why_did_nimble_capture_get_removed_from_the/

== Current Chrome Extensions == 
Nimbus Web inc. (Now owned by MYOB) has additional Chrome extensions listed in the Chrome Web Store that were not affected by the hacking incident in February 2025. https://chromewebstore.google.com/search/nimbus%20web%20inc
