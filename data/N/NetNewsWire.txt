'''<dfn>[https://ranchero.com/evergreen/ NetNewsWire]</dfn>''' is an open source [[feed reader]] for MacOS that is under development.

Developer Brent Simmons keeps “Diaries” on his blog for projects, the [http://inessential.com/2017/06/01/evergreen_diary_1_open_source first entry for Evergreen] went up on 2017-06-01.

== h-feed support ==
There is a github issue to add [[h-feed]] support: https://github.com/brentsimmons/Evergreen/issues/293
* Brent Simmons [http://inessential.com/2018/01/25/evergreen_diary_9_on_indieweb indicated] he would be adding support for h-feed in the 1.0 Alpha version
