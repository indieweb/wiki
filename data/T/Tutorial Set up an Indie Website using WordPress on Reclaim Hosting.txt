== Getting Started on WordPress with Reclaim Hosting ==

This is a step by step tutorial. For mor information on how everything works visit the <a href="https://indieweb.org/User:Jgregorymcverry.com/getting-started-with-WordPress-for-beginners">Getting Started for Beginners Guide.</a>

=== Step One: Set up Shared Hosting ====
      
We will be using <a href="https://reclaimhosting.com">Reclaim Hosting</a> as our shared host. Please register an account and set up a domain. The next step is to install WordPress

{https://www.youtube.com/embed/HzBGb9DQgvk Install WordPress video}

=== Step Two: Uploading an IndieWeb Theme ===

You have three themes you can that are compatible with the IndieWeb tools we can use You can read more about each theme on the <a href="https://indieweb.org/User:Jgregorymcverry.com/getting-started-with-WordPress-for-beginners">beginner guide</a> 

* {https://github.com/dshanske/twentysixteen-indieweb/archive/master.zip 2016 IndieWeb}
* {https://wordpress.org/themes/sempress SemPress}
   
{https://www.youtube.com/embed/taWSmLBDOi0| Install SemPress video}

=== Step Three: Add The IndieWeb Plugins ===
    
{https://www.youtube.com/embed/Do8OyQFXCEc Add IndieWeb Plugins Video}
 
To learn more about each plugin visit the IndieWeb wiki <a href="https://indieweb.org/WordPress/Plugins">Plugin Page</a>

=== Step Four: Add an h-card to your site ===

An h-card tells the rest of the web your wrote your site and are the author

{https://www.youtube.com/embed/oJ0NZLSHB-U Add an h-card video}
 
You can also manually add an h-card
     
{https://www.youtube.com/embed/1Km_6dxNKmo Manually add h-card}
