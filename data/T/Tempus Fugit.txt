'''<dfn>[https://wordpress.org/plugins/tempus-fugit/ Tempus Fugit]</dfn>''' is a [[WordPress]] plugin that provides [[on this day]] functionality as a stand-alone feature which was previously built into the [[Post Kinds Plugin]].

The name is Latin which translates to "time flies".
