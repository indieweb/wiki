This template is used to generate a Nostr profile link with a Nostr icon to the left of the username.

== Blank Template ==
Copy, paste, and fill in your username.

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9; white-space: pre-wrap;">{{nostr-link| npub = }}</pre>

== Example ==
Including this template code:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9; white-space: pre-wrap;">{{nostr-link| npub = t }}</pre>

Will generate:

{{nostr-link| npub = t }}

== Notes ==
Due to licensing and this wiki being public domain, this template currently relies on transcluding the official Nostr account profile image.
