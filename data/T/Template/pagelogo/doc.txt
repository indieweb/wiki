The '''<dfn>pagelogo template</dfn>''' is used to display logo at the top right of a wiki page.

== Blank Template ==
Copy, paste, and fill in the information:

<pre>{{pagelogo|src= |alt= }}</pre>

== Parameters ==
;src : Absolute URL. The logo source. (required)
;alt : Plain text. Brief text description of the logo, shown if the image does not load, and for text browsers like screen readers. See [[alt#Guidelines|guidelines]] for good alt text. (optional but highly recommended)

== Examples ==
<pre>{{pagelogo|src=https://indieweb.org/img/indiewebcamp.svg |alt=IndieWebCamp logo}}</pre>

Will generate:
{{pagelogo|src=https://indieweb.org/img/indiewebcamp.svg |alt=IndieWebCamp logo}}
