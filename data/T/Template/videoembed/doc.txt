This template is used to generate embedded video where "link" is typically a direct video file link or [[video]] post (like [[YouTube]]). It is typically used to embed video from sessions into session pages in for [[IndieWebCamp]]s.

== Blank Template ==
Copy, paste, and fill in the information:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9;">{{videoembed|link=video-embed-URL}}</pre>

== Parameters ==
; link: video post URL to direct video file link 
: Archive.org example: https://archive.org/embed/getting-started-with-word-press-indie-web-popup-2020

== Examples ==
Including this template code:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9;">{{videoembed|link=https://archive.org/embed/getting-started-with-word-press-indie-web-popup-2020}}</pre>

Will generate:

{{videoembed|link= https://archive.org/embed/getting-started-with-word-press-indie-web-popup-2020}}
