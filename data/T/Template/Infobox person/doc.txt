This template is used to an display an infobox and [[h-card]] for a person on their wiki user page. It will float to the right and is intended to be included near the top of the user page.

== Blank Template ==
Copy, paste, and fill in your information:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9;">{{Infobox person
|name = 
|url = 
|photo_url = 
|pronouns =
|summary = 
|elsewhere =
|contact =
|chatname =
}}</pre>

If you put this template on your user page, wrap it with <code><nowiki><div class="p-summary"> ... </div></nowiki></code>, then Loqi will respond to "who is username" with your summary and URL.

== Parameters ==
; name : Your name.
; url : Your URL.
; photo_url : Your photo/avatar URL.
; pronouns : Your [[pronouns]]. ''(optional)''
; summary : A short introduction to yourself. ''(optional)''
; elsewhere : other sites, social media silos. ''(optional)''
; contact : your contact info ''(optional)''
; chatname : your nickname in [[discuss|chat]] ''(optional)''

== Examples ==
<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9; white-space: pre-wrap;">{{Infobox person
|name = gRegor Morrill
|url = https://gregorlove.com
|photo_url = https://gregorlove.com/site/assets/files/3473/profile-2016-med.jpg
|summary = gRegor Morrill is a software developer living in San Diego, California.
|elsewhere = [https://twitter.com/gRegorLove Twitter]
|pronouns = he/him/his
|chatname = gRegorLove
}}</pre>

Will generate:
{{Infobox person
|name = gRegor Morrill
|url = https://gregorlove.com
|photo_url = https://gregorlove.com/site/assets/files/3473/profile-2016-med.jpg
|summary = gRegor Morrill is a software developer living in San Diego, California.

|pronouns = he/him/his

}}

As seen on the user page for {{gRegor}}

== Incorporating other templates ==
In the "elsewhere" section you can also use the additional templates

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9; white-space: pre-wrap;">{{facebook-link| username = username  }}
{{twitter-link| username = username  }}</pre>

To add the following:

{{facebook-link| username = yourname }}

{{twitter-link| username = yourname }}
