This template is used to an display an infobox and [[h-card]] for a person on their wiki user page. It will float to the right and is intended to be included near the top of the user page.

== Blank Template ==
Copy and  paste the code with the double brackets in the box below into your user page and fill in your information:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9;">{{Infobox person
|name = 
|url = 
|photo_url = 
|pronouns =
|summary = 
|elsewhere =
|contact =
|chatname =
}}</pre>

If you put this template on your user page, wrap it with <code><nowiki><div class="p-summary"> ... </div></nowiki></code>, then Loqi will respond to "who is username" with your summary and URL.

== Parameters ==
; name : Your name.
; url : Your URL.
; photo_url : Your photo/avatar URL.
; pronouns : Your [[pronouns]]. ''(optional)''
; summary : A short introduction with your name in a complete sentence. ''(optional)''
; elsewhere : other sites, social media silos. ''(optional)''
; contact : your contact info ''(optional)''
; chatname : your nickname in [[discuss|chat]] ''(optional)''

== Examples ==
<pre>{{Infobox person
| name = gRegor Morrill
| url = https://gregorlove.com
| photo_url = https://gregorlove.com/site/assets/files/6268/profile-2021-square.300x0.jpg
| summary = gRegor Morrill is a software developer living in <span class="p-locality">San Diego</span>, <span class="p-region">California</span>.
| chatname = gRegor
| pronouns = he/him/his
}}</pre>

Will generate:
{{Infobox person
| name = gRegor Morrill
| url = https://gregorlove.com
| photo_url = https://gregorlove.com/site/assets/files/6268/profile-2021-square.300x0.jpg
| summary = gRegor Morrill is a software developer living in <span class="p-locality">San Diego</span>, <span class="p-region">California</span>.
| chatname = gRegor
| pronouns = he/him/his
}}

As seen on the user page for {{gRegor}}

{{clear}}

== Incorporating other templates ==
In the "elsewhere" section you can also use the additional templates like [[Template:mastodon-link|mastodon-link]], [[Template:facebook-link|facebook-link]], [[Template:twitter-link|twitter-link]]

<pre>{{mastodon-link| instance = example.com | username = username }}
{{facebook-link| username = username  }}
{{twitter-link| username = username  }}</pre>

which will generate:

{{mastodon-link| instance = example.com | username = username }}

{{facebook-link| username = username  }}

{{twitter-link| username = username  }}

== Making this template mobile friendly ==

<div class="iwc-infobox h-card">
  <p class="iwc-infobox__title p-name"><strong>gRegor Morrill</strong></p>
  <p>
    <a href="https://gregorlove.com" class="external u-url">
      <img class="iwc-infobox__photo u-photo" src="https://gregorlove.com/site/assets/files/6268/profile-2021-square.300x0.jpg" alt="(photo)">
    </a>
  </p>
  <p>Pronouns: <span class="p-pronouns">he/him/his</span></p>
  <p class="p-summary">gRegor Morrill is a software developer living in <span class="p-locality">San Diego</span>, <span class="p-region">California</span>.</p>
  <p>https://gregorlove.com</p>
  <p><a href="/discuss" title="discuss">Chat</a> Nickname: <span class="p-nickname">gRegor</span></p>
</div>

The current template uses inline styles that force it to always float to the right of body text. This causes a narrow river of text to appear down the left side on narrower viewports.

It is not possible to add media queries to inline styles, and MediaWiki doesn’t allow <code>style</code> elements in templates. Therefore, style rules to make this template mobile-friendly would need to be added to the site’s global <code>custom.css</code> stylesheet.

=== Preview ===

An example with classes instead of inline styles is shown here. To make it work, add the following styles to your user stylesheet:

<pre>
.iwc-infobox {
  background-color: #f8f9fa;
  border: 1px solid #c8ccd0;
  padding: 0.25em 1em;
  margin-bottom: 1em;
  max-width: 100%;

  @media (min-width: 560px) {
    float: right;
    margin-left: 1em;
    max-width: 15em;
  }
}

.iwc-infobox__title {
  text-align: center;
}

.iwc-infobox__photo {
  display: block;
  margin: 0 auto;
  max-width: 100%;
}
</pre>

Then reload this page, and resize the window to make it narrower. You will then see this second infocard take up the full-width of the screen when the viewport is less than 560px wide.
