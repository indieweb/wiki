<span class="p-summary">The '''<dfn>alert template</dfn>''' is used to display an yellow alert box for urgent information about a page, e.g. how it should be improved quickly.</span>

== Blank Template ==
Copy, paste, and fill in text of the alert / call to action:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9;">{{alert|Please [[wikify]] this page with better section headers!}}</pre>

Will generate: 

{{alert|Please [[wikify]] this page with better section headers!}}

== Parameters ==
This template accepts one unnamed parameter, which should be the text (wikitext ok) of the alert.
