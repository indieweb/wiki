This template is used to generate the Calendar navigation box at the bottom of related pages. It uses the [[Template:Navbox|navbox]] and [[Template:Navbox-row|navbox-row]] templates.

== Blank Template ==
Copy, paste, and done.

<pre>{{Calendar}}</pre>

== Parameters ==
There are no parameters for this template currently.

== Example ==
{{Calendar}}
