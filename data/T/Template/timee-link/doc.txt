This template is used to generate date and timestamps on wiki pages that link to https://timee.io/ to allow users to more quickly translate dates and times from UTC format to their local time quickly and easily.

== Blank Template ==
Copy, paste, and fill in the appropriate date and time fields.

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9; white-space: pre-wrap;">
{{timee-link
|displaytext= 
|dateutc= 
|starttimeutc= 
|durationminutes= 
}}

</pre>

== Data formats ==
The general format for the fields are as follows:
* <code>displaytext</code> value will be what is displayed
* <code>dateutc</code> should be in YYYYMMDD format
* <code>starttimeutc</code> should be HHMM in UTC time
* <code>durationminutes</code> should be XXX for the number of minutes the session's duration


== Example ==
Including this template code:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9; white-space: pre-wrap;">{{timee-link
|displaytext= 18:30 March 8
|dateutc= 20190308
|starttimeutc= 1830
|durationminutes= 30 
}}
</pre>


Will generate:

{{timee-link
|displaytext= 18:30 March 8
|dateutc= 20190308
|starttimeutc= 1830
|durationminutes= 30 
}}
