This template is used for including photos on [[events]] and other pages where you want them to be explicit <code>u-photo</code> microformat properties of the container h-* object.

== Blank Template ==
Copy, paste, and fill in the information:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9;">{{photosrcalt|src=http://example.com/pic.jpg|alt=Photo of something awesome}}</pre>

== Parameters ==
;src : Absolute URL. The photo source. (required)
;alt : Plain text. Brief text description of the photo, shown if the image does not load, and for text browsers like screen readers. See [[alt#Guidelines|guidelines]] for good alt text. (optional but highly recommended)
