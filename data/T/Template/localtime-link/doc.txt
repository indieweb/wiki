The '''<dfn>localtime-link</dfn>''' template creates a link to a page that translates a date and time into various timezones for easier remote attendance. The timezone translation is one of [[Meetable]]'s features and is handled by https://events.indieweb.org.

== Blank Template ==
Copy, paste, and fill in the information:

<pre>
{{localtime
| label = 
| date = 
| time = 
}}
</pre>

== Parameters ==
Note the specific format for date and time parameters.

; label
: The text displayed in the link
; date
: date in YYYYMMDD format
; time
: time in HHMM format

== Example ==
Including this template code:

<pre>{{localtime
| label = 18:30 October 27
| date = 20231027
| time = 1830
}}
</pre>
