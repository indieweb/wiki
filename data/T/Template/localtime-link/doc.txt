<span class="p-summary">The '''<dfn>localtime-link</dfn>''' template creates a link to a page that translates a date and time into various timezones for easier remote attendance. The timezone translation is one of [[Meetable]]'s features and is handled by https://events.indieweb.org.</span>

== Blank Template ==
Copy, paste, and fill in the information:

<pre>
{{localtime-link
| label = 
| date = 
| time = 
| tz = 
}}
</pre>

== Parameters ==
Note the specific format for date and time parameters.

; label
: The text displayed in the link. This can be left out if you only want to show the <code>time</code> (see [[#Display_only_the_time|example]] below)
; date
: Date in YYYYMMDD format
; time
: Time in HHMM or HH:MM format
; tz
: Timezone name from the [https://www.php.net/manual/en/timezones.php supported list]
: Examples: Europe/London, America/New_York, America/Los_Angeles

== Examples ==
Including this template code:

<pre>{{localtime-link
| label = 18:30 October 27
| date = 20231027
| time = 1830
| tz = Europe/London
}}
</pre>

Will generate:

{{localtime-link
| label = 18:30 October 27
| date = 20231027
| time = 1830
| tz = Europe/London
}}

=== Display only the time ===

If you only want to show the time as the link text, you can omit the <code>label</code>. This can be useful in a compact event day schedule. See it in practice on [[2026/Düsseldorf/Schedule]].

<pre>{{localtime-link
| date = 20260427
| time = 11:20
| tz = America/Los_Angeles
}}</pre>

Will generate:

{{localtime-link
| date = 20260427
| time = 11:20
| tz = America/Los_Angeles
}}
