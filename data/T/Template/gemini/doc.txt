<span class="p-summary">This template is used to generate a [[Gemini protocol]] link using a web proxy.</span>

== Blank Template ==
Copy, paste, and fill in the Gemini URL.

<pre>{{gemini| url = }}</pre>

== Example ==
<pre>{{gemini| url = gemini://tilde.pink/~brennan/}}</pre>

Will generate:

{{gemini| url = gemini://tilde.pink/~brennan/}}
