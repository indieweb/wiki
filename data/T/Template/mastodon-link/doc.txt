This template is used to generate a Mastodon profile link with a Mastodon icon to the left of the username.

== Blank Template ==
Copy, paste, and fill in your instance and username.

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9; white-space: pre-wrap;">{{mastodon-link | instance = | username = }}</pre>

== Example ==
Including this template code:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9; white-space: pre-wrap;">{{mastodon-link| instance = mastodon.technology| username = djmoch }}</pre>

Will generate:

{{mastodon-link| instance = mastodon.technology | username = djmoch }}

== Notes ==
Due to licensing and this wiki being public domain, this template currently relies on transcluding the official Mastodon.Social favicon.
