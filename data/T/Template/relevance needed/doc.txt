This template is used to indicate the [[definition]] for an article needs to be updated to include [[relevant to the IndieWeb wiki|IndieWeb relevance]]. It should be placed at the top of the page.

Using this template will also add the page to category [[:Category:Definitions_needing_IndieWeb_relevance|Definitions needing IndieWeb relevance]].

== Blank Template ==
Copy and paste:

<pre>{{relevance_needed}}</pre>

== Parameters ==
This template has no parameters currently.

== Examples ==
The warning that this template displays:

{{relevance_needed}}

See pages in [[:Category:Definitions_needing_IndieWeb_relevance]] for examples.
