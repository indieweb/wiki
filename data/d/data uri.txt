'''<dfn>data uri</dfn>''' is a syntax for embedding file data (usually images) in HTML tags.

It can be used as a way to "inline" content to a page.

== IndieWeb Examples ==
* {{jacky}} uses data URIs to embed his PGP key on every page as well as the [[buttons]] for the IndieWeb.
