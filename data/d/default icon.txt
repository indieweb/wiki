'''<dfn>default icon</dfn>''' is an [[icon]] used to represent typically a person or author that lacks their own icon or photo, in a [[reply-context]], or [[responses]] like [[comments]] or [[likes]], or new accounts on [[silo]]s that have not yet uploaded a photo.

== IndieWeb Examples ==
=== Aaron Parecki ===
{{aaronpk}} uses a gray on gray minimal abstract person icon to represent the icon of responses without one. E.g. one of the bookmarks on [https://aaronparecki.com/2020/12/03/1/indieauth-2020#responses this post]

https://aaronparecki.com/assets/images/no-profile-photo.png 

== Silo Examples ==
=== GitHub ===
[[GitHub]] uses a blocky two color pixelated square as a default icon
