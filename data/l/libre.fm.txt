'''<dfn>[https://libre.fm/ libre.fm]</dfn>''' is a free and open source [[scrobbling]] service (since 2009) that is IndieWeb [[friendly]]. 

It is run by [[User:Mat.tl|Matt Lee]] and has more than 380,000 users and a database of more than 347 million scrobbles. Which is a lot.

[https://libre.fm/donate.php Donations are welcome], including tax-deductible donations via [[Open Collective]]. 

== IndieWeb Friendly ==
Libre.fm supports several IndieWeb building blocks, e.g. on profiles:
* [[rel-me]]
* [[h-card]] — e.g. https://indiewebify.me/validate-h-card/?url=https%3A%2F%2Flibre.fm%2Fuser%2Fmattl
...
