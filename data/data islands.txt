'''<dfn>data islands</dfn>''' are defunct technique for adding DRY-violating invisible data (an [[antipattern]]) to HTML pages (AKA '''<dfn>sidecar metadata</dfn>'''), first introduced then abandoned by Microsoft as XML Data Islands, and more recently, by Facebook and Google as (different) JSON(LD) embedded in HTML.

== History ==
* 199x [[Microsoft]] introduces XML Data Islands
* 2011-07-06 Microsoft drops support for XML Data Islands (e.g. in IE)[http://tantek.com/2011/188/t6/ie10-drops-conditional-comments-please-add-history][http://blogs.msdn.com/b/ie/archive/2011/07/06/html5-parsing-in-ie10.aspx]
* 2015-01 [[Google]] adds support[https://twitter.com/kevinmarks/status/557619286114451456] for "embedding" JSON(LD) Data Islands, likely unaware of the history.[http://tantek.com/2015/020/t2/google-reinvents-microsoft-xml-data-islands]
* 2015-05-12 [[Facebook]] launches [[Instant Articles]][http://www.cnet.com/news/facebook-launches-instant-articles-touting-faster-mobile-load-times/], which use a JSON Data Island specifically for embedding [[GeoJSON]] data[https://developers.facebook.com/docs/instant-articles/reference/map].
