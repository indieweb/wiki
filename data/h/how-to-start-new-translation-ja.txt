= 別の言語でIndieWebCampのWiki記事を作成する方法 =

== はじめに ==
IndieWebCampのWikiページを別の言語で作成しましょう。

「[[other-languages|他の言語（other-languages）]]」にご希望の言語が見当たりませんか？ ぜひ、IndieWebCamp Wikiの翻訳に協力してください！

== Wikipediaのヒントを読む ==
差し当たり、全般的なヒントやアドバイス、コミュニティの慣習については、Wikipediaの「[https://en.wikipedia.org/wiki/Wikipedia:Multilingual_coordination 多言語プロジェクトとしてのウィキペディア]」および「[https://meta.wikimedia.org/wiki/Help:How_to_start_a_new_Wikipedia 新しいWikipediaの始め方]」のページを見てください。

== まずはこのページから ==
まずは「[[stable-pages|安定版ページ（stable-pages）]]」のリストから始めるのがよいでしょう。これらのページは内容が比較的安定しており、細かな修正や編集のみにとどまっているため、英語版との同期を保ちやすくなっています。「[[Special:Watchlist|ウォッチリスト（my watchlist）]]」機能を使って、自分が翻訳したページの更新状況をチェックしておくのがおすすめです。

== ページの命名規則 ==
ページ名：翻訳版のページを作成する際は、元のページ名と同じ名前を使用し、その末尾にハイフン（-）とRFC 3066の言語識別コードを付けます。
例：日本語版の場合、[[Main_Page]] は [[Main_Page-ja]] に、[[why]] は [[why-ja]] になります。

== カテゴリの指定 ==
日本語で作成したページには、ページの最下部に<nowiki>[[Category:Pages in Japanese]]</nowiki>を追加してください。

<nowiki>[[Category:Pages in Japanese]]</nowiki>を追加することで、[[:Category:Pages in Japanese|カテゴリ: 日本語のページの一覧]]に自動的に分類されます。
