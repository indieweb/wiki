'''<dfn>mp-syndicate-to</dfn>''' is a property in a [[Micropub]] request, giving the server the command to syndicate the received h-entry to the specified [[silo]].

Syndication targets known by the server can be obtained by using the configuration query for [https://www.w3.org/TR/micropub/#syndication-targets syndication-targets].
