'''<dfn>mf2util</dfn>''' is a Microformats 2 utility for Python, commonly used to extract common features in comments and reply-contexts. mf2util is intended to be used in concert with mf2py.

The package can be installed with <code>pip install mf2util</code> from [https://pypi.python.org/pypi/mf2util/ PyPI]. Documentation is available at [http://mf2util.readthedocs.org/en/latest/ Read the Docs] and source is hosted on [https://github.com/kylewm/mf2util GitHub]
