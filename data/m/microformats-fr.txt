Les <dfn>microformats</dfn> sont des extensions du HTML pour baliser les personnes, organisations, événements, lieux, billets de blog, produits, critiques, C.V., recettes, etc. Les sites utilisent les microformats pour publier une API standard qui est consommée et utilisée par les moteurs de recherche, agrégateurs et d'autres outils.

== Usage IndieWeb ==
Le Web Indépendant fait un très large usage de : 

* [[h-card-fr|h-card]] pour baliser les profils et auteurs dans les posts
* [[h-entry-fr|h-entry]] pour baliser les [[posts-fr|posts]], [[replies-fr|réponses]], etc.
* [[in-reply-to-fr|in-reply-to]] pour baliser les liens provenant de [[replies-fr|réponses]] aux posts originaux.
* [[xfn-fr|XFN]] pour les relations, et la consolidation d'identité (rel=me), utilisée par [[IndieAuth]]
