'''<dfn>microlight</dfn>''' is a blogging engine built on PHP, SQLite, and IndieWeb principles of POSSE. Microlight supports the following post types: note, article, photo, video, reply, like, bookmark, and repost. Microlight is developed and maintained by [[User:Tomgardiner.me|Tom Gardiner]].

== Requirements ==

* PHP 7+ (as anything below is no longer supported by the PHP group)
* GD (for [[micropub media endpoint]]), cURL (for [[micropub]] and [[webmentions]]), and PDO SQLite (for post storage/management via database) extensions for PHP
