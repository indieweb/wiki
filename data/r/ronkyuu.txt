'''<dfn>[https://github.com/bear/ronkyuu ronkyuu]</dfn>''' is a Python open source [[project]] that is being designed to help with [[Webmention]] requirements for static sites.

== Features ==
* Webmention endpoint discovery via HTTP Link header, HTML <<nowiki/>link> tag and HTML <<nowiki/>a> tag
* RelMe endpoint discovery
* Target verification
* Webmention Sending
* HTML scanning for possible Webmentions
* tools that help with link and rel handling
