この '''<dfn>style guide</dfn>'''（スタイルガイド）は、IndieWeb の用語を使用する際の一連の推奨事項です。

このページを使用して、執筆スタイルや専門用語・フレーズの曖昧さ回避について議論したり、他の人々が IndieWeb のアイデアやプロジェクトについて書いている内容との一貫性を保つために、用語に関する質問を行ったりしてください。

== The Indie Web ==
* 名詞としての「the Indie Web」。
* イベント名の「IndieWebCamp」は1単語（スペースなし）です。
* 「IndieWeb」は固有名詞です。これは indieweb.org のようなウェブサイト、IndieWebCamp のようなイベント、および microformats を含む様々な技術を包含する IndieWeb ムーブメントを指します。
* 「indieweb」は、「IndieWeb の、またはそれに関連する」を意味する形容詞です。
* 「indie web」は、独立したウェブ（independent web）を表現する一般的な用語です（元々は [[timeline#1997|1997年に言及されました]]）。
[https://chat.indieweb.org/2013-09-01#t1378064957000000 IRC での要約より]

== An Indie Website ==

すなわち、Indie Web の一部であるサイトのこと。

== Webmention ==
* プロトコルとしての ''[[Webmention-ja|Webmention]]'' は固有名詞であり、大文字で始めるべきですが、CamelCase（WebMention）にはしません。
* 個々の人が受け取る個別の ''webmention'' は固有名詞ではありません。
* 複数の ''webmentions'' を受け取ることはできますが、固有名詞としての ''Webmention'' は複数形にはなりません。

== ブレインストーミング・セクション ==
=== indieweb.tld ===
コミュニティとして、イベントや indieweb.org ウィキのような組織的サイトには IndieWeb および IndieWebCamp を使用するよう努めています。
==== IndieWeb イベント ====

コミュニティイベントのためにウェブサイトを開設する場合、トップレベルドメインに indie web という用語を使用することを歓迎します。Homebrew Website Club (hwc) についても同様です。

* 例
** https://2019.indieweb.org/ - IndieWeb イベント登録
** https://indiewebcamp.nl/ - オランダでの IndieWebCamp
** https://hwclondon.co.uk/ - ロンドンでの Homebrew Website Club ミートアップ

==== IndieWeb サービス ====

サービスに名前を付ける際は、トップレベルドメインに IndieWeb を使用するよりも、そのサービスが解決する問題にちなんだ名前を付けるのが最善です。

コミュニティをサポートするためのサービスを構築する場合、indie という言葉の使用を歓迎します。他の分野における indie という言葉の使用の歴史については、ポップカルチャーからの多数の例がある [[indie]] を参照してください。

* 例
** [https://telegraph.p3k.io/ Telegraph] - Webmention を送信するためのサービス。
** [https://IndiePay.me indiepayme] - 支払いを追跡するための分散型台帳。
** [https://aperture.p3k.io/ Aperture] - Microsub ソーシャルリーダー。
** [https://indieseek.xyz/2018/10/25/indieseek-xyz-choosing-the-domain-name/ IndieSeek]
