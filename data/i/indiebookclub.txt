'''<dfn>[https://indiebookclub.biz indiebookclub]</dfn>''' is an app for posting [[books]] you are [[reading]] to your website using [[Micropub]]. It is based on the [[Teacup]] source code and is now open source: https://github.com/gRegorLove/indiebookclub.

== Features ==
* [[Web sign-in|Sign in using your domain name]]
* All posts appear on your indiebookclub profile, so you can use it even if your site does not support [[Micropub]] yet
* The new post UI supports query parameters and can parse the microformats from other [[read]] posts in order to pre-populate the form [https://indiebookclub.biz/documentation#query]
* Uses Micropub JSON syntax [https://indiebookclub.biz/documentation#micropub]
* [[Export]] your posts in HTML+[[microformats]] format [https://dev.indiebookclub.biz/documentation#export]
* ...

More features are being planned and discussed on Github: https://github.com/gRegorLove/indiebookclub/issues
