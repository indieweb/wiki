'''<dfn>icon font</dfn>''' is a technique for embedding vector graphics in HTML, using custom font files.

== Criticism ==
[[SVG]] is a highly recommended instead of icon fonts. Here is [http://blog.cloudfour.com/seriously-dont-use-icon-fonts/ a reasoned explanation why]. If one insists on using fonts for icons - there are reasons for that as well, such as being easy to embed in text -, [https://petermolnar.eu/icon-fonts-fallback-unicode/ they should at least have meaningful Unicode mappings to provide fallback].
