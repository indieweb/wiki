'''''indielib''''' ([https://go.hacdias.com/indielib go.hacdias.com/indielib]) is a Go library maintained by {{hacdias}}. It provides the following utilities:

* Creation of a [[IndieAuth]] server or client
* Creation of a [[Micropub]] server
* Implements an augmented version of the [[post-type-discovery]]
