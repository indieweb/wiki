'''<dfn>[https://go.hacdias.com/indielib indielib]</dfn>''' is a [[Go]] library maintained by {{hacdias}} that implements several IndieWeb building blocks. 

== Features ==
It provides the following utilities:

* Creation of a [[IndieAuth]] server or client
* Creation of a [[Micropub]] server
* Implements an augmented version of the [[post-type-discovery]]
