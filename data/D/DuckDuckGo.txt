'''<dfn>[https://duckduckgo.com DuckDuckGo]</dfn>''' is a search engine.

== !Bang ==
DuckDuckGo has a shortcut feature called !Bang, which provides quick access to various other search services. This is especially useful when using DuckDuckGo as the default search engine of your browser, so you can use the address bar to search things. An overview can be found on https://duckduckgo.com/bang.

The [https://duckduckgo.com/bang?q=indieweb !indieweb] Bang can be used to search this wiki.

== Other Features ==
* Shows [[favicon]] image beside URL, e.g. https://duckduckgo.com/?q=site%3Atantek.com+harry+august&atb=v37-4__&ia=web [https://duck.co/help/features/favicons]
