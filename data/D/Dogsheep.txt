'''<dfn>[https://dogsheep.github.io/ Dogsheep]</dfn>''' is a project to build tools for pulling personal data from different sources into [[SQLite]] databases.

See also this talk on Personal Data Warehouses https://simonwillison.net/2020/Nov/14/personal-data-warehouses/ which describes Dogsheep and the underlying Datasette tool in detail.
