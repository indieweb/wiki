'''<dfn>email hosting</dfn>''' is a service for receiving and sending [[email]] on the internet.

== Self-hosting ==

=== what you need ===

* [https://en.wikipedia.org/wiki/Fully_qualified_domain_name a fully qualified domain name ( FQDN )]
* a static IP
* [https://en.wikipedia.org/wiki/Reverse_DNS_lookup a reverse dns] entry for your server, which should match the mail server header

Optional, for security:

* [https://en.wikipedia.org/wiki/DomainKeys_Identified_Mail DomainKeys Identified Mail (DKIM)] entry in dns
* [https://en.wikipedia.org/wiki/Sender_Policy_Framework Sender Policy Framework (SPF)] entry in dns

=== Software bundles ===
* [http://www.iredmail.org/ iRedMail]
* [https://www.zimbra.com/open-source Zimbra Collaboration Open Source Edition]

=== Howtos ===
* [https://petermolnar.eu/linux-tech-coding/debian-lightweight-mailserver-postfix-dovecot-dspam Lightweight, secure, database-free, spamfiltering mail server with Postfix, Dovecot, openDKIM and dspam on Debian 7]
