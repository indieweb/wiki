'''<dfn>Firefox</dfn>''' is a free, open source web browser made by [[Mozilla]].

== Hello ==
{{main|Firefox Hello}}
Firefox Hello is a [[WebRTC]] based feature for doing browser to browser calls, audio, video or both, which you can setup with a hyperlink as part of an indieweb [[communication]]/[[contact]] page.

== Firefox OS ==
{{main|Firefox OS}}
Firefox OS is an open source [[mobile]] operating system developed by Mozilla, based on Linux and the Gecko web rendering engine.
