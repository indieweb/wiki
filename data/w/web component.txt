== IndieWeb Examples ==

=== capjamesg ===

{{capjamesg}} has made a few web components on his site.

* <code>recommend-firefox</code>: Adds a message that recommends Firefox if the reader is not using Firefox to read the page (https://jamesg.blog/2023/12/12/recommend-firefox/).
* <code>commandk.js</code>: A search overlay (https://jamesg.blog/2023/02/13/announcing-commandk-js/).
* <code>stories.js</code>: A [[story]] layout (https://jamesg.blog/2023/05/21/stories-js/).

He also uses a web component from Zach Leatherman:

* <code>seven-minute-tabs</code>: Add tabs to a web page (https://github.com/zachleat/seven-minute-tabs).
