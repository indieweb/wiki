'''<dfn>webmention.js</dfn>''' is a common name for a Javascript library for displaying comments and likes from the webmention.io service.

There are a few different things by this name:

* https://github.com/bcomnes/webmention.js by {{bret}} (unmaintained)
* https://github.com/PlaidWeb/webmention.js by {{beesbuzz.biz}}
