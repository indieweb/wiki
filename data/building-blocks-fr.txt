== Mentions ==
La capacité pour une page web de montrer qu'une autre l'a reliée est un bloc de construction indieweb vital. Parmi d'autres objets,  cela aide le logiciel à passer le [[SWAT0-fr|SWAT0]].

Les technologies de mention :

* [[pingback-fr|pingback]]
* [[webmention-fr|webmention]]
* [[salmon-fr|salmon]]

== Connexion ==

== Web Actions ==

Une "web action" est l'interface et l'expérience-utilisateur de produire une action spécifique et discrète, sur l'ensemble du web, provenant d'un site vers un autre site ou application.

* [[webactions-fr|webactions]]

== Prévisualisation de Lien ==

Une prévisualisation de lien est ce que différents silos de réseaux sociaux (par ex. Facebook, Twitter, Google+) affichent sous les notes et commentaires qui ont une ou plusieurs URLs, en extrayant l'information provenant du contenu de ces URLs et en affichant un titre/résumé/photo pour ces URLs sous la note/commentaire posté.

* [[link-preview-fr|Prévisualisation de Lien]]
