[http://indiewebcamp.com/2015/Germany IndieWebCamp DÃ¼sseldorf] last weekend was a great success, with more than 30 participants coming from all over the world! The event spurred some cool new tools and [http://indiewebcamp.com/2015/Germany#Blog_Posts a bunch of great blog posts]. Aaron Parecki [http://aaronparecki.com/articles/2015/05/11/1/ introduced] an awesome new interface for writing long-form articles, built in to his micropub client, Quill. Now any site that supports posting via micropub can use a beautiful [https://quill.p3k.io/editor Medium-inspired editor] to write posts. Many of the hack day projects involved adding Webmention and Micropub support to a bunch of different platforms -- Jekyll, Drupal, phpADNSite, [https://github.com/bastianallgeier/kirby-webmentions Kirby CMS], and more!

Go enthusiasts got a boost this week as {{vendan}}, a relative newcomer to the IRC channel, has been churning out indieweb libraries left and right, including a new [https://github.com/andyleap/microformats microformats2 parser for go].

Kevin Marks's new mf2 parser feature called &quot;rel-urls&quot; was adopted by mf2py and [https://github.com/tommorris/mf2py/blob/master/CHANGELOG.md#026---2015-05-06 released in version 0.2.6]. &quot;rel-urls&quot; is a new field in the JSON output that contains detailed properties for each rel url. It's useful for things like feed discovery and XFN.<br />

Known released [https://github.com/idno/Known/releases/tag/v0.7.8.5.1 a new version], and the Bridgy team [https://github.com/snarfed/bridgy/issues/406 continues] [https://github.com/snarfed/bridgy/issues/407 to grapple] with Facebook's API 2.x changes (which is proving to be a bit of a moving target).<br />
