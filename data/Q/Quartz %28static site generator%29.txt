'''<dfn>Quartz</dfn>''' is an open source ([https://github.com/jackyzha0/quartz GitHub repository]) [[static site generator]] written mainly in TypeScript, some [[JavaScript]], and relies on [[Node.js]]. 

It is marketed as a free alternative to [[Obsidian]] Publish, allowing Markdown content to be shared as a [[static site]]. Quartz is usually geared towards people who want a [[digital garden]], personal knowledge base or write notes with an emphasis on bi-directional linking. 

Some features of Quartz include:

* Full-text search
* Graph view
* Backlinks
* Different flavours of [[Markdown]]
* OxHugo ([[emacs]]) and [[Roam Research]] compatibility
* Supporting [https://quartz.jzhao.xyz/features/obsidian-compatibility Obsidian’s core features]

See [https://quartz.jzhao.xyz/features/ Feature List] for more.

In May 2026, Quartz 5 was released with some significant changes, for example:

* Plugins are now independent packages
* Configuration is now in [[YAML]] instead of TypeScript

See [https://quartz.jzhao.xyz/getting-started/whats-new What's New in Quartz 5] for more.
