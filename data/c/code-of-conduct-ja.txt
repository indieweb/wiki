== 尊重（Respect） ==
__TOC__
IndieWebは、独立したクリエイターの創造性とコラボレーション、そして彼らがIndieWebにもたらす人々、文化、意見の多様性を認識し、称賛する、意図的にポジティブなコミュニティです。

IndieWebの各スペースは、性別（トランスジェンダーの状態を含む）、性的指向、年齢、障害、健康的状況、人種および／または民族、カースト、国籍、宗教（または無宗教）、身体的外見、政治、イデオロギー（現実世界または技術的）、またはソフトウェアの好みに関わらず、すべての個人を尊重して扱うことに基づいた包括的な環境です。

私たちは、個人の意見よりも「尊重ある振る舞い」を重視します。

尊重ある振る舞いには以下が含まれます：
* 思いやりを持ち、親切で、建設的かつ協力的であること。
* 屈辱的、差別的、嫌がらせ、憎悪に満ちた言動、不必要または文脈を無視した性的（身体部位）表現、身体的脅迫を伴う振る舞い、発言、テキスト、[[chat-names|チャット名]]、および画像を避けること。
* 判断に迷う場合は、決めつけずに誰かに尋ねること。

IndieWebは、特権を持つ人々の居心地の良さよりも、<span id="prioritizes_marginalized_people">社会的に不利な立場にある（マイノリティの）人々の安全を優先</span>します。IndieWebの主催者は、以下の内容に関する苦情に対して、対応を行わない権利を有します。
* 「逆」差別（「逆レイシズム」、「逆セクシズム」、「シスフォビア」など）
* 「放っておいてほしい」「あっちへ行って」「この件についてはあなたと議論しない」といった、境界線（パーソナルスペース）に関する妥当な意思表示。
* あなたが「心地よい」と感じない「トーン（口調）」でのコミュニケーション。
* 人種差別的、性差別的、シスジェンダー至上主義的、あるいはその他の抑圧的な振る舞いや前提に対する批判。

<small>（社会的に不利な立場にある人々の安全に関する上記の段落およびリストは、XOXOが [https://2018.xoxofest.com/conduct CC-BY ライセンス] で提供しているものを参考にしています。）</small>

== 平和的な解決 ==
私たちは、ピア・ツー・ピア（個人間）の議論、フィードバック、そして修正が、より強く、安全で、より歓迎的なコミュニティを築く助けになると信じています。

もし誰かが失礼な振る舞いをしているのを見かけた場合は、その行為を止めるよう、敬意を持って促すことを推奨します。コミュニティの他のメンバーも、コミュニティを尊重ある場所に保ちたいと考えており、あなたのそのような行動を歓迎するはずです。

もし失礼な振る舞いを経験し、（いかなる理由であれ）敬意を持って対応したり解決したりすることが難しいと感じた場合は、すぐに主催者（オーガナイザー）に報告してください。あなたが失礼だと感じたこと、脅威に感じたこと、あるいは単に「嫌だ（icky）」と感じたことなど、何でもお聞かせください。私たちは耳を傾け、問題の解決に努めます。

== 過ちへの謝罪 ==
もし自分自身が失礼な振る舞いをしてしまったことに気づいたり、それを指摘されたりした場合は、自分の言葉と行動に責任を持ち、適切に謝罪してください。完璧な人間はいませんし、善意のある人でも間違いを犯すことはあります。重要なのは、その間違いにどう対処し、将来的に繰り返さないようにするかです。

== 処置 ==
主催者が、イベント参加者が失礼な振る舞いをしていると判断した場合、主催者は警告なしの退場や、返金なしでのイベントからの除外を含む、適切と判断されるあらゆる措置を講じることができます。

主催者として、私たちは紛争を平和的に、かつコミュニティにとってポジティブな形で解決することを目指します。しかし、あらゆる状況を予見することは不可能です。したがって、主催者の判断として失礼な振る舞いをする人物に退去を求めることが最善であると判断した場合は、そのように行動します。

== 問題の解決方法 ==
IndieWebのイベントにおいて、以下のポリシーに合致しない慣行を経験していると信じる場合、あるいは何らかの形で嫌がらせを受けていると感じる場合は、すぐにイベントの主催者またはイベント指定の行動規範対応担当者に連絡してください。

チャットやイベントにおいて、上記のポリシーに合致しない慣行を経験していると信じる場合は、すぐに信頼できる知人（ally）に連絡してください。コミュニティの誰もが問題を解決し、私たちの高い基準を維持する権利を持っています。

== ライセンス ==
IndieWeb.orgに投稿されている他のすべてのコンテンツと同様に、この行動規範はCC0の下でパブリックドメインとして公開されています（詳細は [[IndieWeb:Copyrights|Wikiの著作権ページ]] を参照）。

== オープン・コード・オブ・コンダクト ==
以下の手順で、この IndieWeb 行動規範をあなた自身のコミュニティで再利用できます：
# 全体をコピー＆ペーストする（CC0ライセンスにより可能です）。
# 「IndieWeb」という名称をあなたの組織名に置き換える。
# 組織特有のカスタマイズが必要であれば適宜行う。
# 任意で、出典として indieweb.org/code-of-conduct を記載しても構いません。
# 他の人が再利用しやすいよう、あなたの行動規範もCC0ライセンスで公開することを検討してください。
# [[discuss|チャット]]でぜひ教えてください！ 私たちの [[#Similar_Codes_of_Conduct|類似の行動規範リスト]] に追加したり、あなたがどのような改善を加えたかを見たりするのを楽しみにしています。
