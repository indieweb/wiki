'''<dfn>command line interface</dfn>''' refers to the use of text-based commands in a terminal application to take actions such as install or update software, and is part of setting up most [[CMS]]s and running [[SSG]]s; due to difficult to remember and easy to typo commands, typically for web developers, and not recommended for any general audience IndieWeb tools or software.

Instead, please consider building web apps and other visual interfaces any IndieWeb [[projects]] intended for general audiences.
