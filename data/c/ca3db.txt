'''<dfn>ca3db</dfn>''' is a library for permanently storing multiple versions of user avatars found on social networks and websites.

https://github.com/aaronpk/ca3db

== IndieWeb Examples ==
=== Aaron Parecki ===
Aaron uses ca3db to archive avatars from comments received via [[webmention]] so that all images are served via https, and will never disappear even if the image is removed from the original URL.
