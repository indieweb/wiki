'''<dfn>[http://open.litespeedtech.com/mediawiki/ OpenLiteSpeed]</dfn>''' is an open source version of the [[LiteSpeed]] web server software that is a "high-performance, open source alternative to [[Apache]] that retains many Apache-compatible features"[http://www.litespeedtech.com/].

== Issues ==
=== lack critical Apache features ===
OpenLiteSpeed:
* Does not "read apache config files"
* No <code>mod_security</code> support
* Lacks "Full .htaccess support"
