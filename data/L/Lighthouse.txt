'''<dfn>Lighthouse</dfn>''' is a [[Webmention]] server built by {{jacky}} to provide a simple means of self-hosting one's Webmentions.

The source code for the project is available at [https://git.jacky.wtf/indieweb/lighthouse git.jacky.wtf]. It's written in Rust and meant to be distributed as a single binary to make ease of deployment simpler.

== Features ==
* Handles receiving and sending of Webmentions for multiple tenants (users)
* Allows for adding and removal of sites to collect Webmentions for
* Defines multiple parsed forms of the received Webmentions for later retrieval
