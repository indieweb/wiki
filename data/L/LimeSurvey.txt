'''<dfn>[https://www.limesurvey.org/ LimeSurvey]</dfn>''' is an open source survey tool also available as a service with a decent [[UI]], that can be self-hosted on an IndieWeb site.

== How to ==
=== How to self host ===
See https://manual.limesurvey.org/Development_overview

== IndieWeb Examples ==
=== tonz ===
{{tonz}} has [https://chat.indieweb.org/dev/2022-01-26/1643175925138600 been using Limesurvey self-hosted]: <blockquote cite="https://chat.indieweb.org/dev/2022-01-26/1643175925138600">It works well, but has a convoluted interface to create questionnaires. In a previous company we worked well with the Limesurvey team in Germany to create specific new question types.</blockquote>
