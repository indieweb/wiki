= Sites avec Portabilité de Données =
Ci-dessous une liste de sites correspondant aux critères suivants :

# ils vous laissent publier des choses sur le web
# ils vous laissent retrouver '''tout''' le contenu que vous avez posté d'une façon relativement aisée (avec des outils tiers, si nécessaire)

(Note : pouvoir sauvegarder vos données provenant de ces services signifie que vous pouvez faire la transition pour faire tourner votre propre site. [[POSSE-fr|POSSE]] est un meilleur moyen pour que la version canonique de vos contenus demeure sur votre site et soit distribuée sur les autres services.)

== Support ==
=== Disqus ===
[http://help.disqus.com/customer/portal/articles/472149-comments-export Détails]

=== Flickr ===
Flickr a une [https://secure.flickr.com/services/apps/tags/backup gamme disponible d'outils de backup].

=== Instagram ===
[http://instaport.me/ Instaport.me] vous laisse exporter vos photos Instagram.

=== Moveable Type ===
[https://github.com/niallkennedy/Movable-Type-WordPress-export Export vers Wordpress].

=== Tumblr ===
L'API Tumblr API vous laisse exporter tous vos posts.

Il existe une [http://staff.tumblr.com/post/286303145/tumblr-backup-mac-beta application Mac en beta pour télécharger les blogs Tumblr] et une  [http://tumblr2wordpress.benapps.net/ application web pour exporter au format Wordpress].

[http://smarterware.org/8026/why-not-tumblr Gina Trapani] nous alerte sur l'utilisation de Tumblr du fait du manque de fonctionnalité export.

=== Wordpress.com et Wordpress ===
Oui. [http://en.support.wordpress.com/export/ Détails].
