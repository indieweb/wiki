'''<dfn>[https://caddyserver.com/ Caddy]</dfn>''' is an HTTP/2 web server, built for designers, bloggers and developers, with a simple configuration and automatic [[TLS]].

Caddy can provide out of the box support for things such as:
* Automatic HTTPS using [[Let's Encrypt]]
* Markdown rendering
* FastCGI
* Site search
* Static site generation via Hugo
* Site deployment via git
* File manager

Configuration is done via [https://caddyserver.com/docs/caddyfile Caddyfile] and is meant to be simple.
