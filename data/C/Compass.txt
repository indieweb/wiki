'''<dfn>[https://github.com/aaronpk/Compass Compass]</dfn>''' is a location tracking server that stores data in flat files and provides a map view and API to integrate location data with a personal website.

== IndieWeb Examples ==

* {{aaronpk}} uses Compass to continuously track his location, geotag his posts on his website, and log [[bicycle|bicycle rides]]. More info: https://aaronparecki.com/gps/
* {{jeremycherfas}} uses Compass to store location data and recently transferred the endpoint [https://www.jeremycherfas.net/blog/found-my-way-back-to-compass from Dreamhost to Hetzner].
