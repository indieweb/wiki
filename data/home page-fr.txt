= Page d'accueil =
Que devrait-il y avoir sur votre page d'accueil de site indie web ?

== Démarrer ==
Si vous [[getting started-fr|démarrez juste]] :
* [[profile-fr|profil]] / information de contact avec une [[hcard-fr|hCard]]
** exemples d'informations de contact / info à propos uniquement : aralbalkan.com, ebdesign.se, vieux (pre-2012) adactio.com
* [[How_to_set_up_web_sign-in_on_your_own_domain-fr|support de connexion-web]]

== Flux ==
Ensuite, il est tout à fat populaire d'avoir : 
* des [[feed-fr|flux]] de vos posts les plus récents et autres activités.
** des exemples de flux mixés/composés : tantek.com, nouveau (après 2012+) adactio.com
** des flux séparés : aaronparecki.com

== Découverte ==
Votre page d'accueil est l'URL que vous partagez avec les personnes, par conséquent il est utile de la paramétrer avec : 
* [[discovery-fr|discovery]] de ce que vous avez d'autre sur votre site

== Précédemment ==
Parmi les blogueurs des années 2000s il était populaire d'avoir aussi sur votre page d'accueil : 
* [[blogroll-fr|blogroll]]

== Fonctionnalités de Connexion ==
Une fois connecté sur votre site, il pourrait être utile d'avoir : 
* [[feed reader-fr|lecteur de flux]] - flux des posts des autres que vous suivez
** exemples de silo d'une telle page d'accueil une fois connecté : [[Facebook-fr|Facebook]], [[Twitter-fr|Twitter]], [[Tumblr-fr|Tumblr]] - connecté dans le tableau de bord
