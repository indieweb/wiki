* {{artlung}} has removed major tools he was previously using and migrated things and removed a lot of older code which was a maintenance burden
** RSS Reader changed
*** moved from Tiny Tiny RSS to FreshRSS https://www.freshrss.org/
*** Self-hosted, PHP+MySQL, so far very excellent and am using ReadKit on iPad https://readkit.app/
** Link management system changed
*** had a self-hosted homegrown PHP+MySQL which was an old link blog, plus my old delicious links, plus old pinboard links, which partly ingested links from Twitter back in the day
*** New tool is Linkding https://linkding.link/ and am processing my 21673 links. Currently 7010, 14662 to go, posting via a simple POST via the REST API with a simple token auth
*** I am using https://www.pikapods.com/ PikaPods is on-demand app hosting, very straightforward and inexpensive to run - under $2/month for an on-demand instance
*** https://gist.github.com/artlung/15cb9acdf720e5ede9254bd247ccc59b is the code I wrote to submit the links to the REST API
