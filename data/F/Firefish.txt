'''<dfn>[https://joinfirefish.org/ Firefish]</dfn>''' (formerly <dfn>Calckey</dfn>) was a fork of [[Misskey]], a microblogging  [[ActivityPub]] server implementation with an emphasis on features that people enjoy using. 

The new name was introduced with the 1.0 release in July 2023. By March 2024, the main project website domains were offline and only security fixes were being released.

More:
* [https://joinfirefish.org/blog/en/introducing_firefish/ Introducing Firefish]
