<dfn>[http://www.fabfile.org/ Fabric]</dfn> is a python library and command-line tool useful for streamlining deployment and system administration tasks.

=== IndieWeb Users ===
* {{kylewm}} uses this [https://github.com/kylewm/redwind/blob/master/fabfile.py fabfile] to simplify deployment to [https://kylewm.com kylewm.com]. ''fab deploy'' transmits the latest git revision and restarts the uwsgi server.
