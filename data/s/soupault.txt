'''<dfn>[http://soupault.net/ Soupault]</dfn>''' is a [[static site generator]] by [https://baturin.org/ Daniil Baturin], written in OCaml and contained in a single executable. Soupault does not force the use of traditional [[templates]], <abbr title="Domain-Specific Language">DSL</abbr> bracket expressions (such as <code>{{ }}</code> and <code>{% %}</code>) and [[frontmatter]]. Instead, it uses CSS selectors to specify elements to allow built-in widgets, [[Lua]] plugins and external programs to transform markup in a fashion akin to [[microformats]] parsing. Additionally, [[HTML]] is treated as a first-class language capable of logic through the use of ''augmented HTML''.

While soupault itself does not implement any format on its own other than HTML, any shell command can be configured as a preprocessor—it be cmark for [[Markdown]], [[pandoc]] for [[reStructuredText]], AsciiDoctor for [[AsciiDoc]], or custom shell scripts.
