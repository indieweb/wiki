'''<dfn>[http://soupault.net/ Soupault]</dfn>''' is a [[static site generator]] by [https://baturin.org/ Daniil Baturin], written in OCaml and contained in a single executable. Soupault does not make use of [[templates]], <abbr title="Domain-Specific Language">DSL</abbr> bracket expressions (such as <code>{{ }}</code> and <code>{% %}</code>) and [[frontmatter]]. Instead, it uses CSS selectors to specify elements to allow [[Lua]] plugins and external programs to access data in a fashion akin to [[microformats]] parsing. Additionally, [[HTML]] is treated as a first-class language capable of logic through the use of <q>augmented HTML</q>.

While soupault itself does not implement any format on its own other than HTML, any shell command can be configured as a preprocessor—it be cmark for [[Markdown]], [[pandoc]] for [[reStructuredText]], AsciiDoctor for [[AsciiDoc]], or custom shell scripts.
