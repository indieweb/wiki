This template is used to generate "See also <nowiki>[[link]]</nowiki>." It is typically used at the top of page section to point to related articles.

== Blank Template ==
Copy, paste, and fill in the information:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9;">{{see_also|wiki page|optional label}}</pre>

== Parameters ==
; wiki page : The wiki page name as it appears in the URL. It may include an anchor to a specific section of the page
; optional label : Optional label to display for the link. This is useful when linking to an anchor within a page.
== Examples ==
Including this template code:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9;">{{see_also|Atom#IndieWeb_Examples|Atom Indieweb Examples}}</pre>
