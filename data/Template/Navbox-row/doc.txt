This template is used to generate a row for use inside a navigation box.

== Blank Template ==
Copy, paste, and fill in the information:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9;">{{Navbox-row
|title = 
|content = 
}}</pre>

== Parameters ==
; title : The title to appear in the first column of the row.
; content : The content to appear in the second column of the row.

== Examples ==
Refer to the [[Template:Navbox|navbox]] template for an example of how the templates work together.
