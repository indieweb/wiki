This template is used to generate "Main article: <nowiki>[[link]]</nowiki>." It is typically used at the top of page section to point to a longer page on the topic.

== Blank Template ==
Copy, paste, and fill in the information:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9;">{{main|wiki page|optional label}}</pre>

== Parameters ==
; wiki page : The wiki page name as it appears in the URL.
; optional label : Optional label to display for the link.
== Examples ==
Including this template code:

<pre style="line-height: 1.4em; padding: 1em; border: 1px solid #ddd; color: black; background-color: #f9f9f9;">{{main|personal-domain|Personal domain}}</pre>

Will generate:
{{main|personal-domain|Personal domain}}
