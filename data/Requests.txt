<dfn>[http://docs.python-requests.org/en/latest/ Requests]</dfn> is a small, human-friendly, Python library for working with HTTP requests, written by [http://www.kennethreitz.org/ Kenneth Reitz]. The python standard library provides many (all?) of the same functions, but is more awkward and error-prone to use.
