'''<dfn>[https://github.com/owncast/owncast OwnCast]</dfn>''' is a self-hosted [[livestreaming]] application that can be used to livestream from your own website.

== IndieWeb Examples ==
* Matt Steele 2020-08-12 [https://steele.blue/indieweb-wedding-livestream/ Indieweb Livestreaming your Wedding with Owncast]. Description of using [[OwnCast]] with [[POSSE]] to [[Zoom]].
* {{aaronpk}} occasionally livestreams his 3D prints at https://printer.aaronpk.tv and other content at https://live.aaronpk.tv using OwnCast
* KmikeyM livestreams using OwnCast at https://live.kmikeym.com
* [[2021/Düsseldorf|IndieWebCamp Düsseldorf 2021]] was livestreamed with OwnCast on indieweb.live
* {{jacky}} streams whatever he's in the mood off at https://live.jacky.wtf and uses [https://docs.datarhei.com/restreamer/ restreamer] to broadcast to [[Twitch]] as well.
** As of 2022-08-27, I've been using [[User:Jacky.wtf#streamd|streamd]] to manage interactivity with my site from Owncast

== IndieWeb Friendly ==
* 2020-12-22 {{aaronpk}} suggested using [[IndieAuth]] for login [https://aaronparecki.com/2020/12/22/8/indieauth] [https://github.com/owncast/owncast/issues/464#issuecomment-749655495]
* 2022-07-09 [https://github.com/owncast/owncast/releases/tag/v0.0.12 Version 0.0.12] shipped support for claiming a chat account using IndieAuth
