'''<dfn>[https://omnibear.com/ Omnibear]</dfn>''' was an open source browser extension for posting text notes, replies, bookmarks, and likes to your website using [[Micropub]] created by [[User:Keithjgrant.com|Keith J. Grant]].

== Status ==
* [https://chat.indieweb.org/2024-11-24/1732469290464900 Since 2024-11], {{aciccarello}} is talking with Keith and working on fixing Omnibear to work with the latest Chrome and Firefox add-on APIs and download sites. If you want to help please ping <kbd>aciccarello</kbd> in [[discuss#dev|#dev on the chat]]. For reference, the GitHub repos:
** [https://github.com/omnibear/omnibear Omnibear browser extension code]
** [https://github.com/omnibear/omnibear-site omnibear.com website code]

== Features ==
IndieWeb features that Omnibear supports:
* publishing posts:
** [[note]]
** [[reply]]
** [[bookmark]]
** [[favorite]]
** [[repost]]
* along with
** [[tags]]
** [[slugs]]
** Identifying sites that support [[webmentions]] (icon when responding/reacting to posts)

You can right-click on an any [[h-entry]] to reply, like, or repost it. You may also interact with posts on twitter.com and facebook.com the same way.
