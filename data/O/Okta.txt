'''<dfn>[https://developer.okta.com Okta]</dfn>''' is a service that handles authentication and user registration for apps, and is also a [[sponsor]] of several IndieWeb events.

== IndieWeb Examples ==
* {{aaronpk}} uses Okta to log in to his website without a password since [https://aaronparecki.com/2017/10/04/23/passwordless-logins 2017-10]
* Micah Silverman built an IndieAuth server using Okta at [[2019/Demos||IndieWeb Summit 2019]]
