The '''<dfn>[https://openheart.fyi/ Open Heart Protocol]</dfn>''' describes a way to send a single [[reactji|emoji]] as [[reaction]] to a [[post]] used by several [[IndieWeb]] community members.

== Custom Element ==
[https://github.com/dddddddddzzzz/open-heart-element Open Heart Element] is a custom [[JavaScript]] script which can be included on a page to support <code>&lt;open-heart&gt;</code> elements to allow submitting [[reactji]].
