'''<dfn>Yarns Indie Reader</dfn>''' or '''<dfn>Yarns</dfn>''' for short, is an [[reader]] that is built for use with [[WordPress]] as a plugin. It supports [[microformats]] and [[micropub]].

Built by [[User:Jackjamieson.net|Jack Jamieson]] and announced on his site on [http://jackjamieson.net/yarns-indie-reader/ 2018-02-08]. It's based in part on {{acegiak}}'s [[WhisperFollow]] with inspiration by [[Woodwind]].

== Development ==
The WordPress plugin is not yet available in the main repository (yet), but can be downloaded from [https://github.com/jackjamieson2/yarns-indie-reader Github]. [https://github.com/jackjamieson2/yarns-indie-reader/issues Issues can be filed here] as well.

== Installation ==
Instructions for installation can currently be found here: http://jackjamieson.net/yarns-indie-reader/
